## Summary

Around April 21, 2026, Anthropic quietly removed Claude Code from its $20/month Pro subscription tier. The change happened without a press release or changelog notice.

## Key Facts

- Previous docs: "Using Claude Code with your Pro or Max plan"
- Updated docs: "Using Claude Code with your Max plan"
- Existing Pro/Max subscribers appear unaffected
- Per Anthropic's Amol Avasare: "running a small test on ~2% of new prosumer signups"
- GitHub Issue filed: #45236 "Breaking Change: Claude Code CLI Removed from Pro Plan Without Notice"
- Likely migrating to Teams tier, Enterprise, or a standalone premium add-on
- Community reaction: significant frustration, cited as ammunition for local model advocates
