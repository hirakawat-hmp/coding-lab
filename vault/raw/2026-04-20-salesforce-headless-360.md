# Salesforce Headless 360 (Raw)

Source: https://www.publickey1.jp/blog/26/salesforceapiclimcpsalesforce_headless_360.html
Also: https://www.itmedia.co.jp/aiplus/articles/2604/17/news118.html
Fetched: 2026-04-20

## Overview
Announced at TDX 2026 developer conference. Makes all Salesforce functionality accessible to AI agents rather than just human users through traditional web interfaces.

## Access Methods
- APIs
- CLI (command-line interface)
- MCP (Model Context Protocol) servers

## Key Features
- "Skills for Coding Agents" specifically for AI development
- Covers full lifecycle: development, testing, evaluation, deployment, experimentation, monitoring, operations
- Works across CRM, data warehouse, Slack integration

## Significance
Traditional web interfaces designed for humans are "hindrances to AI agents." Headless 360 allows developers to bypass Salesforce's UI entirely, enabling AI agents to autonomously access Salesforce capabilities.
