# Warp is now open-source

- Source: https://www.warp.dev/blog/warp-is-now-open-source
- Published: 2026-04-28
- Author: Zach Lloyd

## Summary

Warp terminal/AI coding tool transitioned to open-source (AGPL license). Uses "Oz" orchestration platform for agent-powered community contributions. OpenAI is the founding sponsor.

- Extended support for OSS AI models (Kimi, MiniMax, Qwen)
- New customization: basic terminal to full IDE
- Settings file for programmatic control and cross-device portability
- Humans managing agents at scale = new model for open-source contribution
