# Raw: OpenSSL 4.0.0 Release

Source: https://github.com/openssl/openssl/releases/tag/openssl-4.0.0
Fetched: 2026-04-15

Release Date: April 14, 2026

## Breaking Changes

- SSLv2 and SSLv3 support removed
- Engine support removed (`no-engine` always present)
- ASN1_STRING is now opaque (API const qualifiers)
- Global cleanup: libcrypto no longer uses atexit()
- Removed deprecated functions: ERR_get_state(), ERR_remove_state()
- Removed legacy tools: c_rehash script, msie-hack option
- Darwin 32-bit and PowerPC support dropped

## New Features

- Encrypted Client Hello (ECH) — RFC 9849 for enhanced TLS privacy
- Post-quantum cryptography: ML-DSA-MU digest support
- cSHAKE function support (NIST SP 800-185)
- Key derivation: SNMP KDF and SRTP KDF
- FFDHE in TLS 1.2 (RFC 7919)
- Deferred FIPS self-tests (-defer_tests option)
- Windows static/dynamic VC runtime flexibility
