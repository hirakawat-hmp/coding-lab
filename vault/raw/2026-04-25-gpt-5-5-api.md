# GPT-5.5 および GPT-5.5 Pro が API で公開 — 2026-04-24

source: https://developers.openai.com/api/docs/changelog
fetched: 2026-04-25

## 要点

- OpenAI が 2026年4月24日、GPT-5.5 と GPT-5.5 Pro を API（Chat Completions + Responses API）で公開
- GPT-5.5 は「複雑なプロフェッショナル業務」向け設計
- GPT-5.5 Pro は「より計算負荷の高い問題解決」向け

## 対応機能

- コンテキストウィンドウ：100万トークン
- 画像入力、構造化出力、関数呼び出し
- プロンプトキャッシング（拡張）
- Batch、ツール検索、組み込みコンピュータ使用
- ホスト型シェル、Apply Patch、Skills、MCP
- ウェブサーチ

## 変更点

- 推論努力デフォルト：「中」
- 拡張プロンプトキャッシュのみサポート
