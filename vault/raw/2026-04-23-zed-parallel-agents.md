# Zed Parallel Agents — 発表

Source: https://zed.dev/blog/parallel-agents
Fetched: 2026-04-23

## 要点

Zedエディターが複数のAIエージェントを並列実行できる「Parallel Agents」機能を発表。

### Threads Sidebar
- 複数のエージェントスレッドを一覧表示、プロジェクト単位でグループ化
- スレッドの停止、アーカイブ、新規作成などの操作が可能
- フォルダとリポジトリへのアクセス制御が可能

### 新しいデフォルトレイアウト
- Threads Sidebar と Agent Panel が左側に配置
- Project Panel と Git Panel は右側に移動
- ユーザーは自由にレイアウトをカスタマイズ可能

### 思想
「AIツールと人間の職人技を組み合わせてソフトウェアを構築する」Agentic Engineeringのコンセプトを重視。
完全自動化と完全手動の間のバランスを重視。
