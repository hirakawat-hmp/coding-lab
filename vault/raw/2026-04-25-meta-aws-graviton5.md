# MetaとAWSが提携、生成AI推論に Graviton5 採用 — 2026-04-25

source: https://www.itmedia.co.jp/news/articles/2604/25/news024.html
fetched: 2026-04-25

## 要点

- Meta と AWS が提携、生成 AI 推論タスクに AWS Graviton5 CPU を大量採用
- Graviton5：3nm プロセス、192コア、前世代比最大 25% パフォーマンス向上、キャッシュ 5 倍
- コア間通信速度 33% 向上
- Meta は GPU（NVIDIA）+ CPU（Graviton）のマルチプロセッサ戦略を採用
- 省エネ目標達成にも貢献（推論は学習ほど GPU を必要としない）
