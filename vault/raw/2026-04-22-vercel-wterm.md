## Summary

Vercel released wterm, a browser-based terminal emulator, as open source. Core developed in Zig compiled to WebAssembly for near-native performance.

## Key Features

- Built with Zig → WebAssembly
- DOM-based rendering: text selection, copy/paste, browser search
- Supports vim, less, htop via Alternate Screen Buffer
- Scrollback history
- Remote shell via WebSocket
- Embeddable with vanilla JS and React examples
- CSS theming, 24-bit color, auto-resize
