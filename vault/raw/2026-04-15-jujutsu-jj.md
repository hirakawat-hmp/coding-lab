## What is Jujutsu?

Jujutsu is a distributed version control system (DVCS) with CLI called `jj`. Modern alternative to Git, inspired by both Git and Mercurial.

## Key Value Proposition

Unusually: "simpler and easier than git, but at the same time, more powerful." Rather than sacrificing simplicity for power, Jujutsu synthesizes the best features from Git and Mercurial.

## Key Features

- Smaller essential toolset, more powerful due to better integration
- Advanced capabilities: simultaneous branch editing, stacked PRs, workspace management
- Git compatibility — run Jujutsu on your own projects while collaborators use Git
- Low-risk adoption — full history retained, can revert to Git anytime

## Workflows

- Squashing, editing, conflict resolution
- Pull request management
- Workspaces and stacked PRs
