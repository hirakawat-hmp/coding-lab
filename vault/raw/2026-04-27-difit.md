# AIと、コードを見ながらやりとりする。ローカルレビューを回すOSS「difit」

source: https://findy-code.io/media/articles/aisaji-yoshiko_pg
fetched: 2026-04-27

## 概要

AI エージェントが生成したコード差分をブラウザで視覚的に確認できるオープンソースツール。
GitHub の PR 画面を再現したローカル環境での差分表示を実現。

## 基本操作

- `difit` — HEAD コミットの差分
- `difit .` — 未コミット差分
- `difit HEAD main` — ブランチ間比較

## 主な機能

- NPM スキルとしてインストール可能（AI エージェント連携）
- 差分に対して直接コメント付与
- 「Copy Prompt」で行番号付きプロンプトとしてコピー
- difit-review スキル: AI によるコード自動レビュー

## 開発背景

著者・よしこ氏が 2025年6月に Claude Code で要件定義書に基づき開発。
リリース翌日で 100 Star、3 週間後に 1000 Star 達成。

## インストール

```bash
npm install -g difit
```
