# "MPC Server" でローカルLLMを"無料AIエージェント"に変えてみる

Source: https://www.gizmodo.jp/2026/05/what_is_mcp_in_local_llm.html
Retrieved: 2026-05-06

## Summary

MCP（Model Context Protocol）サーバーを使ってローカルLLMをAIエージェント化する方法の解説記事。スマートフォンにアプリを追加するように、LLMに新機能を付与できる。

## 主要MCP機能

1. **ファイルシステムアクセス:** Filesystem MCP ServerでAIがPC上のファイルを読み書き・作成・検索可能
2. **Web コンテンツ取得:** Webページアクセス・分析・AI生成ファイル名でローカル保存
3. **専門データアクセス:** 国土交通省地理空間サーバー（不動産データ分析）、draw.io概念図ツール等

## チャットボット→エージェントへの進化

基本MCPサーバー + 専門拡張の組み合わせで、受動的チャットボットから自律調査・文書生成が可能なエージェントへ変革。

## 重要な制限事項

- **複雑な設定:** JSON直接編集が必要
- **高い技術的ハードル:** コマンドラインターミナル操作必須
- **セキュリティリスク:** MCPソースの信頼性確認が不可欠（設定ミスで悪意あるコードが実行される可能性）

## 活用シナリオ

費用ゼロで自律的に動くAIエージェントを構築可能。ただしセキュリティと技術ハードルに注意。
