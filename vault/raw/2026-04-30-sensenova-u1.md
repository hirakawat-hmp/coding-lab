# SenseNova U1 — VAE不要の軽量画像生成AIオープンモデル

source: https://gigazine.net/news/20260430-sensenova-u1-image-generation-ai/
fetched: 2026-05-01

## 概要

中国SenseTime社（センスタイム）が公開したオープンソース画像生成AIモデル。VAE（変分オートエンコーダ）やテキストエンコーダーを不要とする革新的設計。

## スペック

| 項目 | 詳細 |
|------|------|
| パラメーター数 | 80億 |
| 処理速度 | RTX 5090で2048×2048: 前処理0.415s + 生成23.04s |
| 対応機能 | 画像生成・編集・インフォグラフィック・連続画像生成 |
| ライセンス | Apache License 2.0 |
| 配布 | Hugging Face（無料公開） |

## 設計の革新性

従来の画像生成AIはVAEでピクセル空間→潜在空間変換が必要だったが、SenseNova U1は単一モデルで生成処理を完結させる。Qwen-Image-2512やZ-Imageよりパラメーター数は少ないが、ベンチマーク品質スコアは高い。
