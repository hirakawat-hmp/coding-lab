# Google Agentic Data Cloud — 発表

Source: https://www.publickey1.jp/blog/26/googleawssaasaiagent_data_cloud.html
Fetched: 2026-04-23

## 要点

GoogleはGoogle Cloud Next 2026で「Agentic Data Cloud」を発表。
AWS、Azure、Google Cloud上のあらゆるデータソースをAIネイティブなデータレイクハウスに統合。

- **ユニバーサルコンテキストエンジン**：AIエージェントに信頼性の高いビジネスコンテキストを提供
- **クロスクラウド対応**：複数クラウドのデータにシームレスにアクセス
- **Apache Icebergフォーマット**：データをコピーせずに参照
- **ナレッジカタログ**：非構造化データからエンティティ情報を抽出
- **連携**：Databricks Unity Catalog、AWS Glue Data Catalog、Snowflake Polaris など
