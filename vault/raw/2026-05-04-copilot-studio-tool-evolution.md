# Copilot Studio のツール設定が進化している (2026-05-04)

Source: https://qiita.com/Takashi_Masumori/items/9b0e71089d1a166e5623
Published: 2026-05-03

Key points:
- Copilot Studio の3つの新機能:
  1. ツール実行前の確認機能: 登録操作などを実行する前にユーザーに確認を取る設定が追加
     - 確認メッセージはカスタマイズ可能、変数を利用して動的に内容を構成可能
  2. 入力値ごとのユーザー入力要求: 各入力項目について個別に確認の有無とプロンプトをカスタマイズ
     - 入力されない場合のエスカレーション設定も可能
  3. 指示文による曖昧性の対応: 入力項目が満たされても内容が曖昧な場合、AIが詳細情報を聞き返す
     - 指示文とツール入力プロンプトの両方で設定する
- 「より柔軟かつ簡単に」チャットや音声入力から値を設定し、不足や曖昧さを対話的に解決
