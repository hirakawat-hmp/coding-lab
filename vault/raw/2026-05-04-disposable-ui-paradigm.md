# 使い捨てUI：AIエージェントが固定インターフェースを殺す (2026-05-04)

Source: https://japan.zdnet.com/article/35247011/
Also: https://startupfortune.com/ai-agents-are-killing-fixed-app-interfaces-turning-software-into-disposable-pixels/
Also: https://chrisloy.dev/post/2026/02/14/when-interfaces-become-disposable
Published: 2026-05-03

Key points:
- AIエージェントが固定アプリUIをタスク特化の一時的なUIに置き換えるパラダイムシフト
- Disposable Software概念: 耐久資産はスクリーンではなくデータ・パーミッション・ワークフローロジック・API・監査証跡・ドメインモデル
- UIはエッジで生成されユーザーの即時意図に形成され、用が済んだら捨てられる
- Cloudera 2026年予測: エンタープライズ内で短命のAI生成アプリが増加、ガバナンス圧力も高まる
- デザイナーへの影響: 毎画面を描くことから「AIが何を組み立て可能か」の制約定義へ仕事が変化
- デザインシステムは静的コンポーネントよりもAIが組み立てる際のルールセットに
- 「AIが数秒でDBクエリ・可視化・カスタムUIを構築できるなら、固定アプリは作業の起点である必要はない」
