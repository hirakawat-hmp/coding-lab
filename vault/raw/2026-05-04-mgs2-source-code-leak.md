# メタルギアソリッド2 ソースコードとアセットが流出 (2026-05-04)

Source: https://gigazine.net/news/20260503-metal-gear-solid-2-source-code-assets-leaked/
Published: 2026-05-03

OGP Image: https://i.gzn.jp/img/2026/05/03/metal-gear-solid-2-source-code-assets-leaked/00_m.png

Key points:
- 2001年発売「メタルギアソリッド2」PS Vita版HD移植版のソースコードとアセットが4月30日に4chanで流出
- 流出元: PS Vita版移植担当のArmature Studio
- 含まれるもの: MD5チェックサム・ソースファイル・アセット・ビルドデータ
- 規模: 30GB以上のアセット（未使用データを含む）
- PS2版からXbox/PC/Xbox 360/Vita版にいたるマルチプラットフォーム開発過程を記録
- 背景: Armature Studioは2022年Metaに買収、2026年1月に人員削減で閉鎖
  Bluepoint Games（同移植関係）も2026年3月に閉鎖
- 偶然の一致: 流出が4月30日（ゲームストーリーで重要な日付）、小島秀夫氏も同日「4/30」と投稿
