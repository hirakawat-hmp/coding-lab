# Anthropic "Project Deal" — AIエージェント市場実験

**ソース:** https://www.itmedia.co.jp/aiplus/articles/2604/26/news023.html
**取得日:** 2026-04-26
**公開日:** 2026-04-26 07:04 JST

## 記事内容

Anthropic が「Project Deal」という新プロジェクトを発表。AIエージェント技術が実世界の複雑な業務をどの程度処理できるかを検証する実験。

### 実験概要
- 69組の参加者が1週間にわたりテストに参加
- Claude を使用したインタビュー結果から、各チームが独立したAIエージェントを運用
- Slack統合環境で、人間の監督なしにエージェントが自律的に業務を実行

### 主な成果
- 186件の課題に対して、4,000ドルの予算で完結
- Claude Opus 4.5 と Haiku 4.5 の性能差を体験的に確認
- 参加者の46%がこうしたサービスに「満足」と評価

### Anthropicのコメント
「AIエージェントが人間の要望をより正確に理解できれば、企業と個人の間の価値創造が最大化される」
