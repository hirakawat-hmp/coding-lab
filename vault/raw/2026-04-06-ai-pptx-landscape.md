# AI PPTX 生成ランドスケープ

## 商用サービス

### Gamma
- 2025年11月に API v1.0 GA
- Pro アカウント以上で API アクセス可能
- プレゼン、ドキュメント、Web ページ、SNS 投稿を生成
- 非伝統的フォーマット（Web ページ風スライド）に強い

### Beautiful.ai
- 伝統的プレゼンテーションに特化
- パブリック API は未提供（2026年4月時点）
- Smart Template でデザイン自動調整

### SlidesGPT
- Prompt → Outline → Slides の3ステップ
- API 提供あり
- 5分以内で完成

## OSS / プログラマティック

### PptxGenJS
- JavaScript / Node.js
- ゼロ依存、ESM + CJS
- React、Angular、Electron、ブラウザ対応
- HTML テーブル → スライド変換機能
- Claude Code skill の中核技術

### python-pptx
- Python の定番 PPTX ライブラリ
- PPTAgent、ppt-agents 等のエージェントフレームワークで採用
- テンプレート操作 + プログラマティック生成

### dom-to-pptx
- DOM → PPTX のクライアントサイド変換
- CSS スタイル（グラデーション、シャドウ、角丸）を保持
- React / Vue / Svelte 対応

### react-pptx
- PptxGenJS の React ラッパー
- JSX でスライドを宣言的に記述
