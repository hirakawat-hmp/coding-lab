# Google が Anthropic に最大400億ドル投資 — 2026-04-24

source: https://www.cnbc.com/2026/04/24/google-to-invest-up-to-40-billion-in-anthropic-as-search-giant-spreads-its-ai-bets.html
fetched: 2026-04-25

## 要点

- Alphabet（Google親会社）が Anthropic に最大400億ドル（約5.8兆円）投資を発表（2026年4月24日）
- 即時 100億ドルを現金投資（評価額 3500億ドル）、パフォーマンス目標達成で追加300億ドル
- Google Cloud は 5 年間で 5GW のコンピューティングパワーを提供（追加GW も検討）
- Amazon も同月 250億ドルの追加投資を発表済み（累計 1000億ドル以上）
- Anthropic の年間収益ランレート：2025年末 90億ドル → 2026年4月 300億ドル突破

## 背景

Google は既に Anthropic の大口投資家だったが、今回の投資でさらに関係を深める。
Anthropic は Claude シリーズ（Claude Code 含む）の開発元として AI コーディング市場で存在感を高めている。
評価額 3500億ドルは、Anthropic が調達ラウンドのたびに急騰していることを示している。
