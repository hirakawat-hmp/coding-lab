# CC-Canary — Claude Code の品質リグレッション早期検出ツール

source: https://github.com/delta-hq/cc-canary
fetched: 2026-04-25

## 要点

- Claude Code セッションログを解析してパフォーマンスリグレッションを検出
- Agent Skills として 2 コマンドでインストール可能
- ゼロネットワーク要件（完全ローカル動作）
- ~/.claude/projects/ のセッションログのみ読み取り（ユーザープロンプトは 180 文字でトランケート）

## レポート形式

- `cc-canary`：Markdown 形式（GitHub Issue 向け）
- `cc-canary-html`：インタラクティブ HTML ダッシュボード（ダークテーマ、自動ブラウザ起動）

## 追跡指標

- read:edit 比率
- 推論ループ数
- Thinking 削減率
- ターンあたりトークン使用量
- フラストレーション指標

## 判定クラス

HOLDING / SUSPECTED REGRESSION / CONFIRMED REGRESSION / INCONCLUSIVE

インストール：`npx skills add delta-hq/cc-canary`
