## Key Facts

- Version 0.1.0 released on Crates.io (Rust's official package registry)
- Provides WebView API for embedding HTML/CSS rendering in desktop/mobile apps
- Rust-based: memory safety + parallel processing
- Long-term support: LTS versions every 6 months, 9-month support period
- LTS guarantees: no API spec changes + security patches
- Originally developed by Mozilla Research (2012)
- Now under Linux Foundation oversight (since Mozilla 2020 restructuring)
