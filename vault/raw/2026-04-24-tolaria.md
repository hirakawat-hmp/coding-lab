# Tolaria — オープンソース macOS Markdown ナレッジベースアプリ (raw)

source: https://github.com/refactoringhq/tolaria
fetched: 2026-04-24

## 概要

macOS 向けデスクトップアプリ。個人の第二の脳、企業ドキュメント、AI アシスタントのメモリ管理に活用できる Markdown ナレッジベース管理ツール。

## 主な特徴

- **ファイル優先**: プレーンな Markdown ファイルでポータビリティと所有権を確保
- **Git 統合**: 全リポジトリが git ベース、完全なバージョン履歴
- **オフライン対応**: サブスクリプション不要、サーバー依存なし
- **オープンソース**: AGPL-3.0 ライセンス
- **AI 対応**: Claude Code など複数の AI エージェントに対応
