# 人を増やしても減らしてもアウトプットの品質は向上しない (2026-05-04)

Source: https://kosui.me/posts/2026/04/29/knowledge-entropy-and-normative-layer
Published: 2026-05-03

Key points:
- 「人を増やすか減らすか」という二元論を超えた組織出力品質の本質的要因を探求
- 「知識エントロピー」概念: メンバー間の知識分布のばらつき
  - エントロピー増加 → 「すり合わせが必要な情報量」増加 → 「文脈共有時間」増加
  - 文脈共有時間増加 → 意思決定速度低下 → リード層への依存強化
- AI エージェントや自律的人材の導入だけでは解決されない3つの問題:
  1. 一貫性の喪失: 局所最適な判断の累積がUI・データモデルのばらつきを招く
  2. ナレッジの劣化: AI生成知識の矛盾と陳腐化が加速度的に進行
  3. 燃え尽き: 高度に自律的な人材への判断集中が認知負荷を増加させ育成を圧迫
- 解決策: 「自動検出可能な少数の条文」から構成される規範レイヤーの設計
  - 型検査やCI/Linterで自動検出される
  - 過度な詳細化は避け最重要領域に絞り込む
  - Accelerate研究参照: 「生成的」文化を維持しつつ必要最小限の規範を整備
