# 97%のPermission確認を自動化するCoding Agent用OSS「ccgate」

- Source: https://zenn.dev/layerx/articles/20260428-ccgate
- Published: 2026-04-28
- Author: TAK848 (LayerX)

## Summary

ccgate: AIコーディングエージェント（Claude Code / OpenAI Codex）向けPermission確認自動化OSS CLI。

- PermissionRequestフックを傍受し、別LLM（デフォルトClaude Haiku）が判断
- 3段階判定: allow / deny / fallthrough（人間にエスカレーション）
- ルールはjsonnet（自然言語記述）
- metrics / loggingコマンド付き

**実績:** 2026年4月1ヶ月で約2,000件処理。自動化率97%（allowed 1764件・85%、denied 190件・9%、escalated 113件・5%）。応答レイテンシ: 平均約3秒。
