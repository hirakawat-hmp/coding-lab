# Gemini Enterprise Agent Platform — 発表

Fetched: 2026-04-23

## 要点

Googleは「Google Cloud Next 2026」で、AIエージェントの開発・運用・管理に対応する包括的なプラットフォーム「Gemini Enterprise Agent Platform」を発表。

- **Agent Studio**：ローコードでビジュアル開発が可能
- **ADK（Agent Development Kit）**：コード開発用フレームワーク
- **Agent Engine**：実行環境
- **対応モデル**：Gemini、Gemma、Llama、Mistral、Claude、Deepseek など多様なモデルに対応
- **統合機能**：ツールレジストリ、スキルレジストリ、マーケットプレイス
- **セキュリティ**：アイデンティティ管理、可観測性、ポリシー管理を統合
- **接続**：DB、SaaS、ドキュメントストアなど全データソースへアクセス
