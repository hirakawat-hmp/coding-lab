# 2026/6/8 より Google Cloud セッション有効期限のデフォルトが「無期限」から「16時間」に

Source: https://dev.classmethod.jp/articles/google-cloud-session/
Fetched: 2026-05-05

## 変更内容

2026年6月8日から、gcloud auth login で取得した認証情報のセッション期限が無期限から16時間に変更される。

## セキュリティ効果

リフレッシュトークン漏洩時のリスク軽減。これまで漏洩ファイルは「永続的なアクセスを攻撃者に許してしまう」可能性があった。

## ユーザーへの影響

- 16時間ごとにパスワードまたはセキュリティキーでの再認証が必要に
- CI/CDなど非対話型ワークロードは動作停止のリスク

## 対応方法

- Workload Identity Federationへの移行
- 「信頼済みアプリ」への登録による例外設定が可能
- オプトアウト期限：2026年5月29日まで申請すれば現在の設定を維持可能
