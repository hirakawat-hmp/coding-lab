# AWS UAE リージョン復旧状況（2026年4月30日更新）

Retrieved: 2026-05-07
OGP Image: //www.publickey1.jp/2026/aws-middleeaset-region-conflict20260506.png

## 概要

AWSが2月ぶりに中東（UAE）リージョン（ME-CENTRAL-1）の復旧状況を報告。回復には「さらに数カ月かかる見込み」と発表。

## 現状

- 3月のドローン攻撃で3つのアベイラビリティゾーンのうち2つが深刻なダメージ
- 一部のワークロードは継続動作中
- AWSは顧客に対し、アクセス可能なリソースを他リージョンへ移行し、アクセスできないリソースはリモートバックアップから復旧するよう強く推奨
- 課金は停止中

## AWS 中東リージョン

- バーレーン（ME-SOUTH-1）
- UAE（ME-CENTRAL-1）← 被害を受けたリージョン
- イスラエル
- サウジアラビア（計画中）

## 背景

中東紛争（米国・イスラエルのイランへの軍事行動）に起因する地域紛争の余波でデータセンターが攻撃を受けた。
