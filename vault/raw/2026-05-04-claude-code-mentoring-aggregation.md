Published: 2026-05-02

Key points:
- ソニックガーデン社の親方が若手5名の育成情報をClaude Codeで一元管理
- 課題：観察メモ・ふりかえり・日報・育成会議メモが複数ツールに分散
- CLAUDE.mdに業務マニュアルと文体ルールを定義

- Google Docs・社内SNS・スプレッドシートから「取り込んで」→情報統合
- 評価軸に沿った振り返り下書きを自動生成
- 重要：「任せているのは思考の下ごしらえ。AIの出力は必ず事実確認し、観察そのものは親方が担当」
