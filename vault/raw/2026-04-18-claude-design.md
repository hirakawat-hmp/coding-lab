# Claude Design — Anthropic Labs

**Source:** https://www.anthropic.com/news/claude-design-anthropic-labs
**Fetched:** 2026-04-18
**Published:** 2026-04-17

## Summary

Anthropic has launched Claude Design, an innovative tool enabling users to collaborate with Claude to produce polished visual work including designs, prototypes, slides, and marketing materials. The product leverages Claude Opus 4.7 and is available to Claude Pro, Max, Team, and Enterprise subscribers in research preview.

## Key Features

- **Brand Integration:** Automatically builds and applies organizational design systems from codebases and design files
- **Flexible Input:** Accepts text prompts, image uploads, documents (DOCX, PPTX, XLSX), and website captures
- **Refinement Tools:** Supports inline comments, direct text editing, and custom adjustment controls for spacing, color, and layout
- **Collaboration:** Organization-scoped sharing with view and edit permissions
- **Export Options:** Supports internal URLs, folders, Canva, PDF, PPTX, and HTML formats
- **Development Handoff:** Seamlessly packages designs for Claude Code implementation

## Key Points

- Use cases: interactive prototypes, wireframes, design explorations, pitch decks, marketing collateral, code-powered prototypes
- Partners: Canva and Brilliant integrating with the tool
- Datadog reports reducing week-long design cycles to single conversations
- Enterprise admins must enable the feature via Organization settings
- Access available at claude.ai/design
