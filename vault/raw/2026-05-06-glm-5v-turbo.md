# GLM-5V-Turbo: Toward a Native Foundation Model for Multimodal Agents

Source: https://arxiv.org/abs/2604.26752
Retrieved: 2026-05-06

## Abstract

GLM-5V-Turboは、マルチモーダル知覚を補助インターフェースではなく、推論・計画・ツール使用・実行のコアコンポーネントとして統合した基盤モデル。画像、動画、Webページ、ドキュメント、GUIなど異種コンテキストを扱うエージェント応用を対象とする。

## Key Contributions

1. **マルチモーダルネイティブ設計:** 視覚を推論の核に
2. **マルチモーダル訓練手法**
3. **強化学習最適化**
4. **ツールチェーン拡張** (エージェントケーパビリティ)
5. **エージェントフレームワーク統合**

## 性能領域

- マルチモーダルコーディング能力
- 視覚ツール利用
- フレームワークベースのエージェンタスク
- テキストのみのコーディング（競合水準）
