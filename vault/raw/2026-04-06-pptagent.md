# PPTAgent

中国科学院による、リファレンスベースの PPTX 生成エージェントフレームワーク。

## アーキテクチャ

2段階の edit-based アプローチ:

1. **分析フェーズ**: リファレンスプレゼンテーションからスライドレベルの機能タイプとコンテンツスキーマを抽出
2. **生成フェーズ**: アウトラインをドラフトし、選択したリファレンススライドに基づく編集アクションを反復生成

## V2 (2025年12月)

- Deep Research Integration
- Free-Form Visual Design（テンプレート不要の自律デザイン）
- Autonomous Asset Creation
- Text-to-Image Generation
- Agent Environment: サンドボックス + 20以上のツール
- 2026年1月: PPTX エクスポート対応

## Tech Stack

- Python ベース
- Docker サンドボックス（sandbox + host コンテナ）
- Web UI + CLI
- llama.cpp でローカル推論対応
- MCP サーバー統合
- Tavily（検索）、MinerU（PDF パース）との統合
