# GPT-5.5 Instant: smarter, clearer, and more personalized

Source: https://openai.com/index/gpt-5-5-instant
Retrieved: 2026-05-06

## Summary

OpenAI announced GPT-5.5 Instant on May 5, 2026, positioning it as the new default model for ChatGPT, succeeding GPT-5.3 Instant. The model delivers improved reasoning while being faster and more personalized.

## Key Facts

- Released as new default for all ChatGPT users on May 5, 2026
- Available via API as "chat-latest"
- AIME 2025 benchmark: 65.4% → 81.2%
- MMMU-Pro: 69.2% → 76.0%
- New "Memory Sources" capability: shows which prior conversations/files inform responses
- Users can manage and remove memory sources
- Paid users can extend trial period to 3 months
- Future rollout to Free, Go, Business, Enterprise tiers within weeks
