## Summary

Rust 製 AI ネイティブテキストエディタ「Zed」が v1.0 に到達。macOS・Windows・Linux の3プラットフォームに対応。5年間・100万行超のコードを経て GA。

## Key Features

- 複数の並列 AI エージェントを管理（Claude Agent, Codex 等と統合）
- キーストローク単位の編集予測
- カスタム UI フレームワーク GPUI（Rust + GPU シェーダー）
- SSH リモーティング、デバッガー、包括的言語サポート
- Zed for Business: 集中課金・RBAC・チーム管理

## DeltaDB（開発中）

CRDT ベースの同期エンジン。リアルタイムの人間-AI 共同作業コードベース管理を目的とする。

## 位置付け

「1.0 は完成を意味しない」—発表時点で数十万人の開発者が定期利用。
