## Summary

Comprehensive beginner's guide to Claude Code by i-inose on Qiita. Published April 21, 2026. Covers installation through advanced features.

## Key Points

- Core difference: Claude Code executes full workflows (read files, run commands, write code, verify results), unlike GitHub Copilot code completion
- Context window management: most critical concept; limitations require careful management
- CLAUDE.md: project-level memory file loaded at startup
- Plan Mode: Shift+Tab twice to preview planned changes without execution
- Skills & Hooks: automate workflows via slash commands and event-driven shell execution
- MCP: connect external tools (GitHub, Slack, databases)
- Platforms: Terminal CLI, VS Code extension, JetBrains, desktop apps, web (claude.ai/code)
