# 帳票エンジン会社が4年かけたGUIデザイナーを凍結してAIに全振りした話

Source: Zenn https://zenn.dev/yagisan_reports/articles/8a89b77f7106b9
Fetched: 2026-04-28

## 企業概要

デンキヤギ（名古屋の小規模IT企業）が商用の帳票エンジン「yagisan-reports」を2021年から開発。

## 凍結された取り組み

4年以上かけて開発していたGUI帳票デザイナー。当初、商用製品として必須と考えられていた。

## AI転換の背景

2025年初頭のCline導入とClaude Code Maxプランの登場により状況が急速に変化。「プロンプトを数回入力する程度で実用レベルの帳票テンプレートが生成される」能力を確認。2025年8月の展示会でのデモが好評だったことで経営判断が確定。

## 成果と市場反応

展示会で概ね好評。既存ユーザーへのヒアリングでも「AIが実用的なら帳票デザイナーは不要」という一致が得られた。

## 重要な洞察

- 「GUIデザイナーは差別化にならない」という判断
- 将来は「人間がプログラムを書かない」社会構造に向けた開発戦略へ転換
- Agent Skillsという標準規格採用で特定AIツール依存を回避
