# Agents for Financial Services and Insurance

Source: https://www.anthropic.com/news/finance-agents
Retrieved: 2026-05-06

## Summary

Anthropic released ten ready-to-run agent templates for financial services workflows, available as plugins in Claude Cowork and Claude Code, and as cookbooks for Claude Managed Agents.

## Agent Templates (10 Total)

**Research & Client Coverage:**
- Pitch builder, Meeting preparer, Earnings reviewer, Model builder, Market researcher

**Finance & Operations:**
- Valuation reviewer, General ledger reconciler, Month-end closer, Statement auditor, KYC screener

## Key Features

- Claude Opus 4.7 as recommended model (64.37% on Vals AI Finance Agent benchmark)
- Microsoft 365 integration: Excel, PowerPoint, Word, Outlook (coming soon)
- Data connectors from Dun & Bradstreet, Fiscal AI, Guidepoint, IBISWorld, Moody's (600M+ companies)
- 15+ data providers: FactSet, S&P Capital IQ, MSCI
- Managed credential vaults, audit logging, per-tool permissions
- KYC/AML compression from days to minutes

## Enterprise Adoption

JPMorganChase, Goldman Sachs, Citi, AIG, Visa conducting pilot implementations (per Fortune).
Open-source templates available on GitHub Financial Services Marketplace.
