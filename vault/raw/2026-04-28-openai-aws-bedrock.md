# OpenAI models, Codex, and Managed Agents come to AWS

- Source: https://openai.com/index/openai-on-aws
- Also: https://aws.amazon.com/about-aws/whats-new/2026/04/bedrock-openai-models-codex-managed-agents/
- Published: 2026-04-28

## Summary

AWS and OpenAI expanded their strategic partnership on April 28, 2026. Three new offerings on Amazon Bedrock were announced (Limited Preview):

1. **OpenAI Models on Bedrock** — GPT-5.5 and future frontier models accessible via Bedrock APIs with IAM, PrivateLink, guardrails, encryption, CloudTrail logging.
2. **Codex on Bedrock** — The OpenAI coding agent for 4M+ weekly users available on AWS.
3. **Bedrock Managed Agents** — Production-ready OpenAI-powered agents on AWS, using the OpenAI agent harness.

Context: Announced one day after Microsoft-OpenAI exclusive partnership ended. Sam Altman and AWS CEO Matt Garman were interviewed by Stratechery.
