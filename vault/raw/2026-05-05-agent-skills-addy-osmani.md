# Agent Skills — プロダクショングレードのAIコーディングエージェントスキル集

Source: https://addyosmani.com/blog/agent-skills/
GitHub: https://github.com/addyosmani/agent-skills
Fetched: 2026-05-05

## 概要

Google Chrome エンジニアAddy Osmaniが公開した、AIコーディングエージェント向けのプロダクショングレード「スキル」コレクション。
ソフトウェア開発ライフサイクル全体をカバーする20のワークフローと7のスラッシュコマンドで構成。

## コア哲学

シニアエンジニアが実践するワークフロー、品質ゲート、ベストプラクティスを構造化されたスキルとして符号化し、AIエージェントが各開発段階で一貫してこれらの水準を実行できるようにする。

## 主要特徴

- **Anti-rationalization（言い訳防止）**: 各スキルには「後でテストを追加する」などの一般的な言い訳とカウンター主張を記載
- **マルチプラットフォーム対応**: Claude Code、Cursor、Gemini CLI、Windsurf、GitHub Copilot、Kiro など
- **3つの専門ペルソナ**: コードレビュアー（Senior Staff Engineer視点）、テストエンジニア（QA専門家）、セキュリティ監査人

## コンテンツ構成

- 20のコアスキル（Define→Plan→Build→Verify→Review→Ship全ライフサイクル）
- 7のスラッシュコマンド
- 3つの専門エージェントペルソナ
- 4つのリファレンスチェックリスト
