# Cursor 3.0 リリース — 複数 AI エージェントの並行実行を前提とした新 UI

**Source:** https://forest.watch.impress.co.jp/docs/news/2102636.html
**Fetched:** 2026-04-18
**Published:** 2026-04-17

## 概要

AI ネイティブコードエディタ Cursor が 3.0 をリリース。
複数 AI エージェントの並行実行を前提とした「Agents Window」を新設。

## 主要機能

### Agents Window（エージェントウィンドウ）
- VS Code ベースの既存エディタウィンドウとは独立した、マルチエージェント管理専用インターフェース
- サイドバーにアクティブなエージェント一覧（モバイル・Web・デスクトップ・Slack・GitHub など複数ソース）
- リポジトリをまたいだシームレスな切り替え

### 統合開発ツール
- ターミナル・ブラウザ・エディタ・diff ビューアを内蔵
- Git 操作（ステージング・コミット・PR）を統合

### ビジュアルデザインモード
- 内蔵ブラウザで変更箇所を指定してエージェントに指示可能

### クラウド/ローカル柔軟性
- オフライン時もクラウド↔ローカル間でシームレスに引き継ぎ

### v3.1 追加機能
- エージェントペインのタイルレイアウト
- 音声入力（ディクテーション）の信頼性向上
