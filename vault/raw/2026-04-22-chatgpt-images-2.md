## Summary

OpenAI released ChatGPT Images 2.0 on April 21, 2026. Repositions images from standalone creations to "intelligent visual thinking partners."

## Key Features

- "Thinking capabilities": integrates real-time web search and reasoning
- Batch creation: up to 8-10 images simultaneously
- Aspect ratios: 3:1 to 1:3 with multiple output formats
- Enhanced multilingual text rendering (including Japanese)
- Sequential image generation: up to 10 images per prompt
- Available to ChatGPT, API, and Codex users across all tiers
- End-to-end responsible use safeguards
