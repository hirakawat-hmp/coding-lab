# Microsoft 365 Copilot エージェント機能 GA (raw)

source: https://www.itmedia.co.jp/news/articles/2604/24/news068.html
fetched: 2026-04-24

## 概要

Microsoft は 2026-04-22 に Microsoft 365 の Word・Excel・PowerPoint において Copilot のエージェント機能を一般提供開始。

## 各アプリの新機能

### Word

- ドラフト作成だけでなく「対象読者に合わせたツール適用」「コンテンツ改善」「ファイル内での段階的実行」が可能

### Excel

- データ分析・表作成・グラフ生成・モデル構築を自動実行

### PowerPoint

- 既存テンプレートを保持しながら新データでプレゼンテーションを自動更新

## 利用可能層

- Microsoft 365 Copilot（法人）
- Microsoft 365 Premium（法人）
- Microsoft 365 Personal/Family（個人）
