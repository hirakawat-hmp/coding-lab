# AIエージェントを会社で使いたい→「え、セキュリティどうするの？」企業導入アプローチ (2026-05-04)

Source: https://qiita.com/sharu389no/items/ede7d1c0be4a14024857
Published: 2026-05-03

Key points:
- Claude Code・Cursorなどのエージェント導入時のセキュリティ懸念3点
  Q1: AIの行動が見えない → 全操作を記録し改ざん不可形式で保存、異常検知も自動
  Q2: 危険な操作の実行を防止 → 4段階プロンプトインジェクション防御関門 + ルール定義
  Q3: 監査・説明責任 → 44種類の国際規制テンプレートに対応し自動レポート生成
- OSSツール「Aigis」を使った技術的対策
  - インストール: `pip install pyaigis`
  - Apache 2.0の無料OSS、LLM API課金なし
- 2026年特有の対策:
  - MCPツール検査（なりすまし防止）
  - 自己訓練ループによる防御力向上
  - 個人情報の自動マスキング
