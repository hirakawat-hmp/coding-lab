Key points:
- リリース日: 2026年4月29日
- 6月の GitHub Copilot 従量課金移行前の複数の最適化を含む主要アップデート
- 主要新機能:
  1. スキルの独立コンテキスト実行: frontmatterに`context: fork`を指定でサブエージェントとして実行
     - メインの会話コンテキストは一切影響を受けない
  2. リモートコントロール機能: GitHub.com・スマホアプリからCopilot CLIの進捗確認と承認が可能
  3. トークン効率の大幅改善:
     - プロンプトキャッシュ最適化により「93%以上のリクエストがキャッシュ再利用」
     - ツール検索ツール導入で最大20%のトークン削減
  4. Chronicle機能: チャット履歴をSQLiteで構造化、/chronicle:standup でレポート生成
  5. セマンティック検索の拡大: 全ワークスペースで意味ベース検索
  6. OpenAIモデルでWebSocket通信により12%高速化
  7. TypeScript 7.0 Betaサポート強化
  8. Dev Container ロックファイルのデフォルト有効化
