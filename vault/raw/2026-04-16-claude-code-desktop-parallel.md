# Claude Code デスクトップ版を刷新 — 並列エージェント対応でマルチタスクを強化

**Source:** https://www.itmedia.co.jp/aiplus/articles/2604/15/news080.html  
**Date:** 2026-04-15  
**Retrieved:** 2026-04-16

## 記事概要

Anthropic が Claude Code デスクトップアプリを大幅刷新。複数タスクの同時管理と並列エージェント実行に対応。

## 主要ポイント

- **対応プラン:** Pro / Max / Team / Enterprise（Claude API バックエンド経由）
- **サイドバー強化:** アクティブセッション表示・履歴・ステータスインジケーター・プロジェクト整理・フィルタ
- **タスク管理:** キーボードショートカット（Ctrl+; on Windows）でのクイックタスク分割
- **ターミナル統合:** Mac でのローカルセッションおよび SSH 経由リモートマシン接続
- **ファイルプレビュー:** HTML・PDF ファイルのアプリ内表示に対応
- **CLI プラグイン互換性:** デスクトップとクラウドセッション、ローカルマシンアクセス

## 技術的詳細

同時複数タスク実行により「複数の作業を並列で進める」ことが可能に。
新サイドバーがアクティブセッションと利用パターンを整理。
Routines 機能（スケジュール実行）は別途発表。
