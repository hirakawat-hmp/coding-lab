## Summary

Meta Research (Paul Starke, Sebastian Starke) released AI4AnimationPy — a Python framework for AI-driven character animation using neural networks. Python port of the original AI4Animation (Unity-based), eliminating game engine dependencies.

## Key Features

- Entity-Component-System architecture with game-engine-like lifecycle
- Vectorized math library: quaternions, matrices, Euler representations
- Neural network architectures: MLPs, autoencoders, flow matching, codebook matching
- IK solvers, motion modules (root trajectories, joint contacts)
- Import pipelines: GLB, FBX, BVH
- Execution modes: standalone, headless, manual
- Optional real-time rendering via Raylib
- Dependencies: NumPy, PyTorch
