# マネーフォワードのGitHub不正アクセス事件

source: https://zenn.dev/awesome_kou/articles/moneyforward-github-source-leak
retrieved: 2026-05-02

## 概要
2026年5月1日、マネーフォワードがGitHubへの不正アクセスを公表。認証情報の漏えいにより、ソースコードと一部個人情報が流出した可能性がある事案。

## 被害内容
- GitHub認証情報が漏えいし、第三者がリポジトリをコピー
- ビジネスカード保持者370件の名前と下4桁が流出
- ソースコード内に各種認証キー・パスワードが含まれていた
- 本番データベースの直接流出は確認されていない

## 認証情報漏えいの主要経路
1. 開発者PCのマルウェア感染（Stealer型）
2. 公開リポジトリへの誤コミット
3. サードパーティ連携の侵害
4. フィッシング攻撃

## 問題の本質
- 本番カード情報がテストデータとしてリポジトリに混在していた可能性
- 認証情報のハードコード化（シークレット管理サービス未使用）
- Git履歴に残された過去の秘匿情報
