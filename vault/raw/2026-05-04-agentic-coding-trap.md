# Agentic Coding Is a Trap (2026-05-04)

Source: https://larsfaye.com/articles/agentic-coding-is-a-trap
Hacker News: https://news.ycombinator.com/item?id=48002442
Published: 2026-05-03

Key points:
- AIが自動生成したコードを監督するには、そのコードを自分で書けるスキルが必要という矛盾
- AIツール使用で数ヶ月以内に開発者のデバッグスキルが47%低下（著者調査）
- ベンダーロックイン：企業がトークンコストに依存するようになり継続的な経費圧力に直面
- 急速なスキル衰退でジュニア開発者の学習機会が失われる
- 著者が提唱する代替アプローチ：計画・仕様生成にAIを使い実装は手動で行う
- 「レビュー可能な範囲のコードのみ生成し、常に実装プロセスに関与すること」
- "ship computer"の比喩：AIをナビに使い、船を動かすのは自分
