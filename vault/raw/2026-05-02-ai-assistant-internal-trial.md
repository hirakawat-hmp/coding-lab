# 社内向けAIアシスタントを3か月間試験運用してみた

source: https://zenn.dev/mkj/articles/87c3ce556cde90
retrieved: 2026-05-02

## 概要
松尾研究所が2026年2月から、Slack上でAIアシスタント「mbot」を3プロジェクトで試験運用。
nanobotベースで開発、PM業務の負荷軽減を目的。

## 技術スタック
- Claude Code/Codex CLI対応
- Slack統合（メンション応答、スレッド会話）
- GitHub Apps認証
- スケジュール自動実行
- Agent Skills拡張機能

## 活用例
- 週報自動作成：Notion情報から自律的にまとめ
- 論文・ニュース配信：プロジェクト特化パーソナライズ
- コードレビュー：マルチエージェントで複数LLM検証

## アンケート結果（8名回答）
- コードレビュー・資料検索：85.7%の利用率
- 業務効率化評価7以上：100%、9以上：62.5%
- 週30分以上短縮：75%達成
- 全社導入推奨「強く推す」：50%

## 課題
1. セットアップ難度：クラウド環境、各種API設定が複雑
2. コスト：1プロジェクト月1〜2万円
3. セキュリティ：権限管理とデータ破壊リスクの両立
