### 2. PyTorch LightningにShai-Hulud（砂虫）マルウェア混入 — AIトレーニングライブラリへのサプライチェーン攻撃

2026年4月30日にPyPIパッケージ `lightning`（2.6.2・2.6.3）に悪意あるコードが混入。インポート時に14.8MBのマルウェアをダウンロードし、GitHub・AWS・Azure・GCP認証情報を窃取。**Claude Code の `.claude/settings.json` にSessionStart hookを仕込む**手法が使われており、AIツールを使った開発環境そのものが感染経路となる。Semgrepが発見・公開。影響を受けた場合は即時クレデンシャルローテーションが必要。

### 3. Claude Code「OpenClaw」コミット言及で課金ルート変更バグ — HN フロントページに登場

![](https://image.itmedia.co.jp/aiplus/articles/2605/01/ts1648348_202604302_2_w490.jpg)

Gitコミット履歴に「OpenClaw」という文字列があるだけで、Claude Code Max/Teamサブスクリプションを無視してExtra Usage（従量課金）が発生するか、リクエストが拒否されるバグがHNで報告。HERMES.md課金バグと同様の構造。OpenClawはClaude OAuthトークン転用によるAPI課金回避手段として一時期使われており、Anthropicの対策実装の副作用とみられる。
