Anthropic が「1.0〜1.35倍」と案内した Claude 4.7 のトークナイザー変更。実測では技術文書で **1.47倍**、本番コードで **1.45倍** に達した。80ターンの Claude Code セッションで $6.65 → $7.86〜$8.76（**20〜30% 増加**）。CJK 言語はほぼ影響なし（1.01倍）。キャッシュ無効化と組み合わさりコスト増が増幅される点に注意が必要。

### 3. MCP に「システム的脆弱性」— Anthropic は積極的修正を見送り

OX Security が Anthropic の Model Context Protocol（MCP）に6つの重大な脆弱性を開示。5,000 件超のダウンロードと 7,000 以上のサーバーに影響。主な問題：パブリック IP への無制限 AI サービスアクセス・ディレクトリトラバーサル・サンドボックス不備・ツール呼び出しのデータ漏洩リスク。Anthropic は「デフォルト設定は安全だが開発者側が保護を担う」とのスタンスで積極的な修正対応を見送った。

### 4. Google API キー脆弱性：13時間で約900万円請求インシデント

![Google API Key](https://asset.watch.impress.co.jp/img/ipw/docs/2102/509/01_s.jpg)

Firebase に Gemini API を有効化した直後、露出した API キーが悪用され 13時間で €54,000（約900万円）が課金された事案。根本原因：GCP の API キーはデフォルト無制限で、Gemini を有効化すると「非機密」だった Firebase キーが突然 Gemini の認証資格情報になる。Truffle Security は 2,863 件の脆弱なキーを発見。対策：API ごとにキーのスコープを明示制限し、Generative Language API を除外すること。

**関連ページ:** [[wiki/google-apikey-gemini-billing]]
**ソース:** [Google APIキーの脆弱性により13時間で約900万円請求](https://qiita.com/miruky/items/fde2d0747358cd7870d7)

### 5. Claude Codeで実際に起きたセキュリティ事故7選と防止策
