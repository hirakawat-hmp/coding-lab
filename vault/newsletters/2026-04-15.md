### 2. Backblaze がクラウド同期フォルダのバックアップを告知なしで停止

OneDrive・Dropbox・Git フォルダがバックアップ対象外になっていたことが発覚。383GB の OneDrive データが知らぬ間にバックアップされていなかったという報告が HN で 928 pt を集め大議論。2015 年の「すべてのデータを制限なしでバックアップ」という約束から大幅後退。重要な教訓: クラウド同期 ≠ バックアップであり、定期的なバックアップ範囲の確認が必要。

### 7. OpenSSL 4.0.0 リリース — ポスト量子暗号・ECH 対応

OpenSSL 4.0.0 が 2026年4月14日にリリース。SSLv3 完全削除、Engine サポート廃止などの破壊的変更に加え、Encrypted Client Hello (ECH / RFC 9849) とポスト量子暗号アルゴリズム ML-DSA-MU を新規搭載。3.x 系からの移行では Engine API 利用者はプロバイダー API への移行が必要。HN で 184 pt。

**関連ページ:** [[wiki/openssl-4]]
**ソース:** [GitHub Release](https://github.com/openssl/openssl/releases/tag/openssl-4.0.0)

### 14. RTX 5060 Ti × 2 でコスパ最高の 32GB VRAM ローカル LLM 環境

### 20. AI 時代に「実装だけ」では足りないと気づいた話
