### 1. GPT-5.5 リリース（コードネーム "Spud"）

OpenAI が新 LLM モデル **GPT-5.5** を ChatGPT および Codex 向けにロールアウト開始。Terminal-Bench 2.0 で 82.7%、SWE-Bench Pro で 58.6% を達成。エージェント的コーディング・コンピュータ操作・データ分析が得意領域。複雑なマルチパートタスクを渡すだけで計画・ツール使用・自己チェック・継続実行を自律的に行う。Plus/Pro/Business/Enterprise ユーザー向け。

### 2. Anthropic Claude Code 品質低下問題の公式ポストモーテム

Anthropic が 2026-03〜04 の Claude Code 品質低下について公式ポストモーテムを公開。特定された 3 つの問題は全て 2026-04-20 までに修正済み:

1. **推論努力レベルのデフォルト変更**（3/4〜4/7）: UI 凍結対策として「高」→「中」に変更したが品質が低下
2. **キャッシュ最適化バグ**（3/26〜4/10）: 推論履歴を「1回だけ」でなく「ずっと」削除し続ける実装バグ
3. **システムプロンプト簡潔化指示**（4/16〜4/20）: 「25語以下に」という指示が評価で 3% 性能低下を引き起こした

**関連ページ:** [[wiki/anthropic-claude-code-postmortem-2026-04]]
**ソース:** [An update on recent Claude Code quality reports](https://www.anthropic.com/engineering/april-23-postmortem)

### 3. Bitwarden CLI サプライチェーン攻撃（Checkmarx キャンペーン）

![Bitwarden CLI 侵害](https://cdn.sanity.io/images/cgdhsj6q/production/65ef8dc5e66260e20fdf13cead82ebd41b705ee6-1018x666.png)

Bitwarden CLI バージョン **2026.4.0** が「Checkmarx サプライチェーンキャンペーン」の一部として侵害。GitHub Actions の CI/CD パイプライン経由で悪意のある `bw1.js` が注入され、GitHub トークンとクラウド認証情報を C2（`audit.checkmarx[.]cx`）に送信する。10M+ ユーザー・5万+ 企業が対象となった可能性あり。Bun v1.3.13 インタプリタを悪用し、ロシアロケールでは自動終了するという設計が特徴的。

**関連ページ:** [[wiki/bitwarden-cli-supply-chain-2026-04]]
**ソース:** [Bitwarden CLI compromised in ongoing Checkmarx supply chain campaign](https://socket.dev/blog/bitwarden-cli-compromised)

### 4. TorchTPU — PyTorch を TPU 上でネイティブ実行

Google が TorchTPU を発表。既存の PyTorch コードを最小限の変更で Google TPU 上で実行できる。3 つのイーガー実行モード（デバッグ・ストリクト・フューズド）があり、フューズドモードでは 50〜100%+ の性能向上を実現。バックエンドに XLA・StableHLO を使用。DDP/FSDPv2/DTensor/MPMD など現代的な分散訓練パターンを全サポート。2026 年中に GitHub リポジトリを公開予定。

**関連ページ:** [[wiki/torch-tpu]]
**ソース:** [TorchTPU: Running PyTorch Natively on TPUs at Google Scale](https://developers.googleblog.com/torchtpu-running-pytorch-natively-on-tpus-at-google-scale/)

### 5. Microsoft 365 Copilot エージェント機能 GA

![Microsoft 365 Copilot](https://image.itmedia.co.jp/news/articles/2604/24/yu_copilot.jpg)

Microsoft が 2026-04-22 に Word・Excel・PowerPoint の Copilot エージェント機能を一般提供開始。Word では対象読者への自動最適化と段階的実行、Excel では自動データ分析・グラフ生成、PowerPoint では既存テンプレートを保持しつつ新データで自動更新が可能になった。「最終判断はユーザーが行う」という Human-in-the-loop 設計を維持。法人向け（M365 Copilot / Premium）と個人向け（M365 Personal/Family）で利用可能。

**関連ページ:** [[wiki/microsoft-365-copilot-agent]]
**ソース:** [Microsoft 365のCopilotエージェント機能が一般提供開始](https://www.itmedia.co.jp/news/articles/2604/24/news068.html)

### 7. Tolaria — オープンソース macOS Markdown ナレッジベースアプリ

Hacker News トップページに登場した macOS 向けオープンソースアプリ **Tolaria**（v2026.4.23）。プレーン Markdown ファイル+Git ベースで、サブスクリプション不要・サーバー依存なし。Tauri + React + Rust 製。Stars ~1,600（2026-04-23 時点）。Claude Code など AI エージェントとの直接統合を想定した設計で、[[wiki/llm-wiki-pattern]] の思想を具現化したツール。AGPL-3.0。

**関連ページ:** [[wiki/tolaria]]

### 9. AI 導入における経営層と現場のギャップ（ITR 調査）

ITR 調査により、大企業の 70% が「AI を活用できている」と回答する一方、実際の導入実績は 38%（32 ポイント差）という AI 導入ギャップが明らかに。「企業を変える AI」（経営層の変革志向）と「楽に使える AI」（現場の効率化志向）の目的のズレが PoC 止まりを生み出す構造的問題を指摘。

### 10. フランス France Titres データ漏洩（最大 1,900 万件）

![France Titres 侵害](https://www.bleepstatic.com/content/hl-images/2026/04/21/Titres.jpg)

フランス内務省傘下の身分証明書発行機関 **France Titres（ANTS）** が 2026-04-15 に発覚したサイバー攻撃によるデータ漏洩を確認。ハッカー「breach3d」が最大 1,900 万件のレコード（氏名・住所・生年月日・メール等）を盗んだと主張し売却申し出。CNIL・ANSSI・パリ検察局が対応中。機関は「ポータルへの不正アクセスには使えない」と表明するも、フィッシング・社会工学攻撃のリスクは残る。

**関連ページ:** [[wiki/france-titres-breach-2026]]
**ソース:** [French govt agency confirms breach as hacker offers to sell data](https://www.bleepingcomputer.com/news/security/french-govt-agency-confirms-breach-as-hacker-offers-to-sell-data/)

### 11. MeshCore 開発チーム分裂：AI 生成コードの透明性問題

世界 38,000+ ノード・100,000+ ユーザーのメッシュ通信プロジェクト **MeshCore** が公式分裂。原因はチームメンバーの Andy Kirby が Claude Code 等の AI 生成コードを秘匿しながら開発を進め、かつ 2026-03-29 に商標を無断申請したこと。Discord 投票でコミュニティの不信任が示された。「AI ツールの使用をチームに開示すべきか」という OSS 界での規範形成に向けた問いを提起した事件。

### 12. ドッガーランド古代 DNA 研究：16,000 年前の温帯林

![ドッガーランド](https://media.wired.jp/photos/69e8550032cd1e26fc974d14/master/w_1600%2Cc_limit/1940x500.jpg)

ウォーリック大学が北海海底の沈んだ陸塊「ドッガーランド」を sedaDNA（堆積古代 DNA）で分析。41 本のコア・252 サンプルから 47 億リードを解読し、**16,000 年以上前**にコナラ・ニレ・ハシバミ等の温帯樹木が存在していたことを確認（従来推定より数千年早い）。さらに 400,000 年前に絶滅とされたサワグルミ属の DNA も検出。氷期の人類にとって温帯林の避難地として機能していた可能性を示す。

**関連ページ:** [[wiki/doggerland-ancient-dna]]
**ソース:** [北海に沈んだ大地「ドッガーランド」](https://wired.jp/article/doggerland-ancient-dna-ice-age-forest-refuge/)
