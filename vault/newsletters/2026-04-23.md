# Daily Knowledge Digest — 2026-04-23

## 要約

Google Cloud Next 2026でGoogleが大規模なAIエージェント基盤発表ラッシュ。OpenAIも同日ChatGPT Workspace Agentsを発表し、エンタープライズAIエージェント競争が激化した一日。セキュリティ面ではFirefox/TorのIndexedDB脆弱性とAppleの通知キャッシュ残留バグの修正が話題に。LLMのOver-editing研究や、27Bで397Bを超えるQwen3.6-27Bリリースなど技術的な進歩も目立つ。

### 4. Google 第8世代TPU「8t」「8i」— 学習と推論を専用チップに分化

学習用「8t」（前世代比3倍性能、2.7倍コスト効率）と推論用「8i」（推論コスト80%削減、新ネットワーク「Boardfly」採用）の2機種を発表。学習と推論のワークロード分化を専用シリコンで対応する戦略。2026年下半期に提供開始予定。

**関連ページ:** [[wiki/google-tpu-8]]

### 6. Qwen3.6-27B — 27Bで397Bを超えるコーディングモデル

Alibabaがリリースした27Bの密なコーディング特化モデル。SWE-bench Verified（77.2）でQwen3.5-397B-A17B（76.2）を上回った。モデルサイズは55.6GB対807GBと14倍の差。1Mトークンコンテキストにも対応。小さいモデルが大きいモデルを超える「効率化の波」が加速している。

**関連ページ:** [[wiki/qwen3-6-27b]]
**ソース:** [Qwen3.6-27B: Flagship-Level Coding in a 27B Dense Model（HN）](https://news.ycombinator.com/item?id=47863217)

### 8. Firefox/Tor IndexedDB 脆弱性（CVE-2026-6770）— プライバシーの根本的破綻

FirefoxおよびTor BrowserのIndexedDBから、全プライベートセッションを横断して同一ユーザーを特定できる安定した識別子が漏洩する脆弱性が報告された。Torが提供する匿名性の核心を破るもので、Firefox 150・ESR 140.10で修正済み。**今すぐアップデートが必要。**

**関連ページ:** [[wiki/firefox-tor-indexeddb-cve]]
**ソース:** [Firefox/Tor IndexedDB Privacy Vulnerability](https://fingerprint.com/blog/firefox-tor-indexeddb-privacy-vulnerability/)

### 9. Apple：削除済みiPhoneメッセージがFBIに抽出されていた問題を修正

![AppleiPhone通知](https://techcrunch.com/wp-content/uploads/2026/04/iphone-pop-up-notifications.jpg)

削除済みメッセージが通知キャッシュに最大1ヶ月残留するバグをAppleが修正。FBIがこの欠陥を利用してSignalの削除済みメッセージを抽出していたことが判明。エンドツーエンド暗号化アプリのプライバシーがOS層の欠陥で崩れる構造的問題を改めて浮き彫りに。

**関連ページ:** [[wiki/apple-signal-notification-fix]]
**ソース:** [Apple fixes bug that cops used to extract deleted chat messages](https://techcrunch.com/2026/04/22/apple-fixes-bug-that-cops-used-to-extract-deleted-chat-messages-from-iphones/)

### 10. LLM Over-editing 研究 — Claude Opus 4.6が最小変更モデルに

AIコード編集モデルが「必要以上にコードを書き換える」問題の実証研究。Claude Opus 4.6がレーベンシュタイン距離0.06で最も変更を最小化。GPT-5.4は0.395と最も過度編集。推論モデルほど過度編集の傾向があり、明示的な指示と強化学習で改善できることが示された。

**関連ページ:** [[wiki/llm-over-editing]]
**ソース:** [Over-editing refers to a model modifying code beyond what is necessary](https://nrehiew.github.io/blog/minimal_editing/)

### 11. Go 1.24のTLSが気づかぬうちにPQC（耐量子暗号）で動作中

Go 1.24からX25519MLKEM768がTLSのデフォルト鍵交換として有効化されている。NIST標準化（2024年8月）後の「Harvest Now, Decrypt Later」攻撃への対策。今の暗号化通信を傍受保存し量子コンピュータで後で復号するリスクに対し、今から備える重要性が説かれている。

**関連ページ:** [[wiki/go-pqc-tls]]
**ソース:** [Go 1.24からのTLS、気づかぬうちにPQCで動いてるらしい](https://zenn.dev/kimifan/articles/pqc-kuwashii)

### 12. Blender Buddy — APIキー不要のローカルBlender AIエージェント

CGMatterがBlender 5.1向けAIエージェントアドオン「Blender Buddy」を無料公開。Llama.cppを使ってローカルで完全動作し、テキスト質問・アクション実行・ビジョンモード・Web検索に対応。要件：16GB RAM、10〜22GBストレージ。プライバシー重視ユーザーへの選択肢。

### 14. AlphaFold 5周年 — 「どの問題を解くか」を問う科学AIの未来

![AlphaFold 5周年](https://media.wired.jp/photos/69e0f21e0efc9e27a24e1170/master/w_2560%2Cc_limit/1352901220)

2020年登場のAlphaFoldが5周年。2億件の予測結果が350万人の研究者に活用され、2024年ノーベル化学賞を受賞。現在はAI co-scientistによる「複数AIが協働して仮説を生成・検証」するモデルへ進化中。「AIが解き方を支援し、科学者はどの問題を解くかに集中できる」という人間とAIの役割分担論が印象的。
