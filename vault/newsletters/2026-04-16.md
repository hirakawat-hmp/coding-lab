# Daily Knowledge Digest — 2026-04-16

## 要約

AI エージェント開発ツールの進化（OpenAI Agents SDK・Claude Code Desktop・Cloudflare CLI）と、AI によるサイバーセキュリティ脅威の高まり（Cal.com クローズドソース化・GPT-5.4-Cyber・Mythos）が今日の主要テーマ。物理世界への AI 展開（Gemini Robotics・日立フィジカルAI・日産ビジョン）と、プライバシー侵害事例（Google/ICE）が並行して注目を集めた。

### 1. Claude Code Desktop 並列エージェント対応

![Claude Code Desktop](https://image.itmedia.co.jp/aiplus/articles/2604/15/yu_claudecode.jpg)

Anthropic が Claude Code デスクトップアプリを刷新し、複数 AI エージェントの同時実行に対応。新サイドバーでアクティブセッションを管理し、Ctrl+; ショートカットでタスクを即座に分割できる。Mac では SSH 経由のリモートマシン接続にも対応し、本格的な並列開発ワークフローを実現。Pro/Max/Team/Enterprise プランで利用可能。

### 3. OpenAI Agents SDK 次世代バージョン — サンドボックス・ハーネス強化

### 4. Microsoft SQL MCP Server — 複数 DB への AI アクセスを統合

![SQL MCP Server](https://www.publickey1.jp/2026/sqlmcpserver-la01.png)

Microsoft が PostgreSQL・MySQL・SQL Server・Azure Cosmos DB などへの同時接続に対応した「SQL MCP Server」をオープンソースで公開。REST API・GraphQL・MCP の3プロトコルを単一設定で提供し、既存クライアントと AI エージェントの両方から同一 DB を操作可能。完全オープンソース（MIT）でオンプレ・AWS・Azure どこでもデプロイ可能。

### 10. Cal.com クローズドソース化 — AI 時代の OSS セキュリティ転換

Cal.com（スケジューリング OSS）がクローズドソース化を発表。理由：AI が OSS コードベースを体系的スキャンして脆弱性を発見・悪用できる時代になった。AI が 27 年前の BSD 脆弱性を発見し数時間でエクスプロイトを生成した事例を引用。Cal.diy を MIT ライセンスで OSS として維持しつつ、本番コードは非公開化。OSS の「透明性」が AI 攻撃の助けになるという新しいパラダイムの登場。

**関連ページ:** [[wiki/cal-com-closed-source]]
**ソース:** [Cal.com Goes Closed Source](https://cal.com/blog/cal-com-goes-closed-source-why)

### 11. Google・ICE データ提供事件 — プライバシーポリシー破りの教訓

![Google Privacy](https://www.eff.org/files/banner_library/google-spy-eye.png)

EFF が告発。コーネル大の PhD 候補が 2024年9月のパレスチナ支持デモに参加後、ICE の行政召喚状に Google が**事前通知なし**でデータ提供（IP・住所・セッション情報）。10年間維持してきた「事前通知ポリシー」を破った形。EFF は両州の司法長官に不正競争行為を申し立て。「プラットフォームのポリシーは変わる」という教訓。

### 14. SATO: Strips as Tokens — SIGGRAPH 2026 採択の 3D メッシュ AI

三角ストリップ着想のトークン順序付けで、アーティスト品質の 3D メッシュと UV マップを同時生成する AI「SATO」。既存手法では乱雑になりがちなトポロジーと UV シームを、UV 境界を明示的にエンコードした連鎖フェイス列表現で自然に整理。三角・クワッドの両メッシュをデコード可能。SIGGRAPH 2026 条件付き採択。

### 17. CRISPR、ダウン症の余分な染色体サイレンシングへの重要な一歩

![CRISPR研究](https://scx1.b-cdn.net/csz/news/800a/2026/gene-editing-tool-show.jpg)

Harvard Medical School と Beth Israel Deaconess Medical Center の研究チームが改良型 CRISPR/Cas9 でダウン症の余分な第21染色体に XIST 遺伝子を挿入し、20〜40% の統合効率でサイレンシングに成功（ヒト幹細胞）。遺伝子単体でなく染色体全体を標的とする初の概念実証として、PNAS に掲載。安全性・動物モデル検証が今後の課題。

**関連ページ:** [[wiki/crispr-down-syndrome]]
**ソース:** [CRISPR takes a bold leap toward silencing Down syndrome's extra chromosome](https://medicalxpress.com/news/2026-04-crispr-bold-silencing-syndrome-extra.html)
