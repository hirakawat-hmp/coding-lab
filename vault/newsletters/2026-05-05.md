### 3. Claude Code WebFetch の内部動作 — Haikuが要約している実態

Claude CodeのWebFetchは「Received 204.4KB」と表示していても、実際にはHaikuが要約した数百文字のみをOpus/Sonnetに渡している。バイパス条件は「信頼ドメイン80以上 ∧ Markdown ∧ 10万文字以下」のAND条件で、多くのサイトは要約経由。コスト削減・Prompt Injection対策・著作権保護が設計意図だが、ユーザーへの開示が不足している。

**関連ページ:** [[wiki/claude-code-webfetch-haiku-internals]]
**ソース:** [Zenn](https://zenn.dev/zhizhiarv/articles/claude-code-webfetch-haiku-summary)

### 4. xAI Colossus GPU 利用率11% — 55万台保有で6万台分しか活用できず

![xAI Colossus](https://media.xenospectrum.com/large_x_AI_Colossus_Data_Center_Compute_Hall_e9905d2af8.webp)

総投資額180億ドル・55万台超のGPUを保有するxAI Colossusだが、MFU（計算効率）は11%にとどまる。Meta（43%）・Google（46%）の3〜4倍の差。急速なハードウェア拡張にソフトウェア最適化が追いつかなかった結果。新社長Michael Nicollsが2ヶ月以内のMFU50%達成を宣言。「GPUを積む≠有効活用できる」という本質的な教訓。

### 10. Transformers Are Inherently Succinct — ICLR 2026 Outstanding Paper

ICLR 2026 Outstanding Paperを受賞した理論計算機科学論文。Transformerは有限オートマトンより「二重指数的に簡潔」、LTL/RNN/SSMより「指数的に簡潔」であることを形式的に証明。副産物としてTransformerの性質検証がEXPSPACE完全（計算論的に非常に困難）であることも示された。少ないパラメータで複雑なパターンを学習できる理論的根拠が初めて確立された。

**関連ページ:** [[wiki/transformers-inherently-succinct]]
**ソース:** [arxiv.org/abs/2510.19315](https://arxiv.org/abs/2510.19315)

### 11. Google Cloud セッション有効期限変更 — 2026年6月8日から「16時間」に

### 14. Microsoft Edge — 全パスワードをメモリ内に平文保存
