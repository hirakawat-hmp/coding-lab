# メタルギアソリッド2 ソースコードとアセット流出事件

![MGS2流出](https://i.gzn.jp/img/2026/05/03/metal-gear-solid-2-source-code-assets-leaked/00_m.png)

2001年発売の名作ゲーム「メタルギアソリッド2」のPS Vita版HD移植版に関するソースコードとアセットが2026年4月30日に流出した。

## 概要

| 項目 | 内容 |
|------|------|
| 流出日 | 2026年4月30日（4chan掲示板） |
| 流出元 | Armature Studio（PS Vita版移植担当） |
| 規模 | 30GB以上（未使用データを含む） |
| 内容 | MD5チェックサム・ソースファイル・アセット・ビルドデータ |

## 背景

- **Armature Studio**: 2022年にMetaに買収、2026年1月に人員削減で閉鎖
- **Bluepoint Games**: 同移植に関与したスタジオも2026年3月に閉鎖
- 両社の閉鎖後に内部データが漏洩したという流れ

## 歴史的価値

PS2版からXbox・PC・Xbox 360・Vita版にいたるマルチプラットフォーム開発の過程が記録されており、ゲーム開発史料として高い価値を持つとも言われる。

## 偶然の一致

ゲームストーリーにおいて「4月30日」は重要な日付。小島秀夫氏が同日「4/30」とXに投稿していたことがネット上で話題になった。

## 示唆

- スタジオの閉鎖・買収後は内部データ管理が手薄になりやすい
- クローズドソースのゲームも長期保存により漏洩リスクが増す
- 著作権者（Konami）の対応が注目される
