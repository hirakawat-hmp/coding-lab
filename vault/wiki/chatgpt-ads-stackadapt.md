# ChatGPT への広告配信——StackAdapt x OpenAI パイロット

![ChatGPT Ads](https://static-www.adweek.com/wp-content/uploads/2025/05/open-ai-data-chatgpt-2025.jpg)

OpenAI の広告パートナーである StackAdapt が、ChatGPT のプロンプト関連性に基づく広告配信プログラムを選定広告主向けに展開していることがリークされたピッチデックで判明（2026年4月20日報道）。

## プログラム概要

| 項目 | 内容 |
|------|------|
| プログラム名 | OpenAI × StackAdapt Limited Pilot |
| 料金モデル | CPM $15–$60 |
| 最低出稿額 | $50,000 |
| ターゲティング | プロンプト関連性（prompt relevance）ベース |
| ピッチデック共有日 | 2026年3月27日 |

## 背景と意義

ChatGPT は長らく広告なしの有料サブスクリプションモデルを維持してきたが、今回のパイロットは OpenAI が収益源の多様化に動いている可能性を示す。広告は「ユーザーが商品やサービスを調査・比較するタイミング」を狙う **発見レイヤー（discovery layer）** として位置付けられ、従来の検索広告との類似性がある。

## 注意点

- パイロット段階（限定的）
- StackAdapt は早期アクセスとして割引プラットフォーム費・管理費を提供
- ChatGPT の検索利用増大に伴い、Google の検索広告モデルへの挑戦になり得る
