# SnapState

AIエージェントワークフロー向けの永続状態管理プラットフォーム。マルチステップのワークフローをセッションをまたいで保存・再開・リプレイできる。

## 概要

エージェントが長時間タスク（数分〜数時間）を実行する際、途中でクラッシュしたり中断したりしても、チェックポイントから再開できる仕組みを提供する。

## 主要機能

| 機能 | 内容 |
|------|------|
| チェックポイント保存 | 最大1MBのJSONペイロードを各ステップ後に記録 |
| 再開 | 最新チェックポイントからワークフローを継続 |
| リプレイ | 履歴全体をステップバイステップで再生し、デバッグ可能 |
| マルチエージェント追跡 | エージェントIDでタグ付けした監査ログ |
| 分析 | 障害パターン検出、パフォーマンス指標 |

## 対応環境

- **SDK**：JavaScript、Python
- **プロトコル**：MCP（Model Context Protocol）対応
- **統合先**：Claude Desktop、Cline など

## ユースケース

1. **長時間バッチ処理**：データ変換パイプラインを途中から再開
2. **マルチステップ調査**：Webリサーチエージェントが中断しても続きから実行
3. **デバッグ**：本番で失敗したワークフローをローカルで再現・調査
