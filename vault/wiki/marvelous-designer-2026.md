# Marvelous Designer 2026.0

衣服専門の 3D モデリング・シミュレーションソフトウェア「Marvelous Designer」の 2026 年第一弾メジャーアップデート。3D ペンシル機能、レーシングツール、EveryWear テンプレートリギングなどを追加。

## 概要

- **開発元**: CLO Virtual Fashion
- **用途**: 衣服・布地の 3D パターン設計・シミュレーション
- **主な利用者**: ゲーム・映像業界の 3D アーティスト、ファッションデザイナー
- **主な競合**: CLO3D（同社別製品）、Clo, Blender（Cloth シミュレーション）

## 2026.0 の新機能

### 3D ペンシル
3D アバター上に直接ラインを描画し、自動的に 2D パターンを生成する直感的な機能。従来の平面パターン設計からの革新的な転換。

### レーシングツール
靴・コルセットなどの穴（アイレット）に沿った複雑なレーシング構造を精密に作成。

### 2D パターン変形中の 3D 形状保持
2D パターンのリサイズ中も 3D ガーメントのジオメトリを維持し、不要な歪みを防止。

### EveryWear テンプレートリギング（ベータ）
衣服タイプに合わせたリグ生成とスキニングを自動化。ゲームエンジン連携（Unity, Unreal Engine）のワークフローを効率化。

### トゥーンシェーダー
スタイライズな可視化のためのトゥーン描画エフェクト。
