# SKILL.md パターン — Claude Code の折りたたみ式マニュアル

## 概要

SKILL.md は、特定タスクの実行手順を定義する Markdown ファイル。
CLAUDE.md がプロジェクト全体のルールを常時ロードするのに対し、
SKILL.md は「呼び出された時だけロードされる折りたたみ式マニュアル」として機能する。

## 構造

必ず YAML frontmatter で始まる:

```yaml
---
name: スキル名
description: いつ・何をするか（1-2文）
---
```

**frontmatter のみ初期ロード** → 実際に使う際に本体が展開される  
→ コンテキストウィンドウを効率的に活用できる

## 記述すべき要素

- スキルの目的と前提条件
- 引数定義（`$ARGUMENTS`）
- ステップバイステップの実行手順
- 期待される出力フォーマット
- 品質基準と人間の判断が必要なポイント

## 実装パターン例

### 記事執筆スキル

```
リサーチ → 構成案作成（承認待ち） → 執筆 → 保存
```
ユーザー確認ポイントを明示的にフロー内に挟む。

### コードレビュースキル

5軸評価をテーブル形式で出力:
- セキュリティ / パフォーマンス / 保守性 / テスト / 破壊的変更

### デプロイスキル

本番環境での実行前に明示的なユーザー承認を必須とする。

## CLAUDE.md との使い分け

| | CLAUDE.md | SKILL.md |
|---|---|---|
| ロードタイミング | 常時 | 呼び出し時のみ |
| 用途 | プロジェクト全体のルール | 特定タスクの手順 |
| コンテキスト消費 | 大（常時） | 小（使用時のみ） |

## 導入のポイント

1. 繰り返しているパターンを1つずつスキル化する（最初から完璧を目指さない）
2. チームで共有・バージョン管理する
3. 引数を明示することで再利用性を高める
