# 政府 AI「源内」（Gennai）

デジタル庁が 2026年4月24日に GitHub でオープンソース公開した日本政府の AI 活用基盤。商用利用も可能。江戸時代の発明家・平賀源内から命名された。

## 概要

| 項目 | 内容 |
|------|------|
| 開発元 | デジタル庁 |
| 公開日 | 2026年4月24日 |
| ライセンス | オープンソース（商用利用 OK） |
| 目的 | 政府内業務での LLM 活用基盤 |

## 特徴と意義

日本政府が自らの AI ツールをオープンソース化したことは、以下の観点で重要：

1. **透明性の確保**：行政 AI の仕組みをオープンにすることで、国民・研究者が監査可能
2. **民間活用の促進**：商用利用 OK により、民間企業・スタートアップがベースとして活用できる
3. **国産 LLM エコシステムの強化**：ガバメントクラウドと連携し国産 LLM の活用を見据える

## 背景

日本政府の「AI 戦略」の一環として行政 DX を加速する動きの一つ。
デジタル庁は以前より行政サービスの AI 化を進めており、源内の公開はその集大成的な位置づけ。

## 命名の由来

平賀源内（1728–1780）：エレキテル（静電気発生装置）の復元など、江戸時代に多分野で革新的な発明・発見を行った人物。日本の「イノベーターの祖」として名高い。
