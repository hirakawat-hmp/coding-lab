# Claude Design で初稿を爆速作成する

Figma を開かずに Claude Design でプロトタイプ・ワイヤーフレーム・LP・スライドを生成する実践ガイド（まさやん著 Zenn book より）。

## Claude Design の基本

Claude Design はプロンプト 1 つでデザイン初稿を生成する AI ツール（Anthropic Labs）。以下のドキュメントタイプに対応:

- インタラクティブプロトタイプ
- ワイヤーフレーム
- ランディングページ
- プレゼンスライド
- マーケティング素材

## 4 つの編集手段

| 手段 | 使いどき |
|------|---------|
| テキストプロンプト | 大きな方向変換、追加要素 |
| UI 直接編集 | テキストや配置の微調整 |
| インラインコメント | 特定箇所への修正指示 |
| Tweaks スライダー | 間隔・色・レイアウトの微調整 |

## 「Decide for me」機能

迷った際の判断を AI に委ねる機能。色・フォント・レイアウトの選択肢で迷ったときに使う。入力フェーズでは AI が必要な詳細情報を先回りして質問してくる。

## デザインシステムの自動構築

テックブログ 1 本からデザインシステム全体が自動構築される。コードベースやデザインファイルをインポートすれば既存のブランドカラー・フォント・コンポーネントを自動適用する。

## MCP コネクタ対応

Claude Design から直接 MCP コネクタを呼び出せる。外部データソースや API と連携したデザインプロトタイプが作れる。

## 実用指標

- 「25 分で週枠 80%」 — 著者による計測値
- 週あたりのデザイン作業時間を 8 割削減できた実績

## Claude Code へのハンドオフ

Claude Design →「実装パッケージング」で Claude Code に渡せる。デザインをコードに落とし込む際の Figma 的な役割を果たす。

## プランガイド

| プラン | 向いているケース |
|--------|----------------|
| Pro | 個人・週数件のデザイン |
| Max | 頻繁なイテレーション・大規模プロジェクト |
