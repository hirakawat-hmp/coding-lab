# EU バッテリー交換義務化——2027年2月より全スマートフォン・タブレット対象

EU が2027年2月18日から、EU 域内で販売されるすべてのスマートフォンとタブレットに対してユーザーが交換可能なバッテリーを義務付ける規制が発効する。

## 主要要件

| 要件 | 内容 |
|------|------|
| 発効日 | **2027年2月18日** |
| 対象 | EU 域内販売のスマートフォン・タブレット全機種 |
| バッテリー交換 | 専用工具なしで交換可能（または必要工具を無償付属） |
| 交換用バッテリー入手可能期間 | 最終販売後 **5年以上** |
| ソフトウェアアップデート | **5年以上** 提供義務 |
| 充電端子 | 2024年以降製造機はUSB-C 必須（別規制） |

## 市場規模

- 対象市場: EU 域内年間 **1.5億台** のスマートフォン + **2,400万台** のタブレット
- 消費者便益試算: 2030年までに **約200億ユーロ** 節約
- 電子廃棄物: 現在の再利用率は約40%——改善が期待される

## 業界への影響

Apple, Samsung など一体型バッテリー設計を採用してきたメーカーは設計変更が必要。現行の「修理しにくい」設計トレンドへの直接的な規制圧力。

「修理する権利（Right to Repair）」運動との整合性も高く、EU の製品持続可能性規制の一環として位置付けられる。
