# ElevenLabs Scribe v2 Realtime

ElevenLabs が公開した最新リアルタイム音声認識（STT）API。約150msのレイテンシーで90言語に対応し、会話エージェントや会議アシスタントのユースケースに最適化されている。

## 主要スペック

| 項目 | 値 |
|------|-----|
| レイテンシー | 約 150ms |
| 対応言語 | 90言語以上 |
| 音声フォーマット | PCM（8〜48 kHz）、μ-law |
| 価格 | $0.28/時間〜（年間ビジネスプラン） |

## 主な機能

- **リアルタイムストリーミング**：WebSocket・REST API 両対応
- **音声アクティビティ検出（VAD）**：正確な音声セグメント分割
- **手動コミット制御**：文字起こし確定タイミングをアプリが制御
- **テキストコンディショニング**：接続リセット後もシームレスに継続
- **次単語予測**：リアルタイム精度を向上

## 性能の強み

- 技術用語・薬品名・固有名詞を含む複雑な語彙に対応
- フィラーワード・間・感情的表現を適切に処理
- 背景ノイズのある低品質音声でも機能
- 多様なアクセント・発音への適応

## 競合比較

公式ベンチマークにおいて Gemini Flash 2.5・GPT-4o Mini・Deepgram Nova 3 を精度で上回ると公表。

## 用途

- 会話 AI エージェント
- 会議アシスタント・議事録
- リアルタイム音声アプリケーション
