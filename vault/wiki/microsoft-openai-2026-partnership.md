# Microsoft–OpenAI パートナーシップ改定（2026年4月）

2026年4月27日、OpenAIとMicrosoftは設立時の独占契約を改定し、OpenAIが複数クラウドプロバイダーを自由に選択できる新体制へ移行した。

## 主な変更点

| 項目 | 旧体制 | 新体制 |
|------|--------|--------|
| クラウド | Microsoft Azure 独占 | 他社（AWS・Amazon等）も選択可 |
| Microsoft の投資義務 | あり | 終了 |
| 技術支援義務 | — | 2030年まで継続 |
| 独占ライセンス（モデル・製品）| Microsoft が保有 | **2032年まで継続** |
| Azure での優先提供 | あり | **継続** |

## 背景

OpenAIはAmazon（AWS）、NVIDIA、SoftBank Groupとのインフラ提携を近年積極的に拡大しており、今回の改定はその戦略の正式化を意味する。Microsoftとの独占収益分配も終了し、複数クラウドの柔軟な活用が可能になった。

## 業界への意義

- AI市場における競争が一層激化
- クラウドプロバイダー間のAI API争奪が本格化
- OpenAIにとっては資金調達・インフラ面の柔軟性向上
- Microsoftはモデルへの独占ライセンス（〜2032年）を維持するため、Azure への実質的な優位は残る
