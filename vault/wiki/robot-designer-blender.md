# Robot Designer — Blender プロシージャルロボット作成アドオン

Alexis Egi氏が開発したBlender用アドオン。Geometry Nodes を活用して、リギング・テクスチャリング済みのロボットキャラクターを**10分以内**で作成できる。Superhive で販売中。

## 主要機能

**Geometry Nodes ベースのプロシージャル設計**
- 800以上のパラメータをスライダーで調整
- 空のシーンから完成品まで完全自動

**リグ**
- Rigify IK/FK リグ
- Mixamo リグ対応

**マテリアル・テクスチャ**
- PBRスマートマテリアル（Cycles + EEVEE 両対応）
- 自動UV展開
- テクスチャベイク（7種類のマップ）
- IDマップ・Substance Painter 対応

**エクスポート**
- FBX、OBJ、GLB
- LOD生成

**プリセット**
- 10種類の手作りプリセット付属
