# Google が Anthropic に最大400億ドル投資（2026年4月）

![Google が Anthropic に最大400億ドル投資](https://techcrunch.com/wp-content/uploads/2025/03/google-anthropic-investment.jpg)

Alphabet（Google 親会社）が 2026年4月24日、AI スタートアップ Anthropic への投資として最大 **400億ドル（約5.8兆円）** を実施すると発表した。

## 投資の構造

- **即時投資**：100億ドルを現金で投資（評価額 **3,500億ドル**）
- **条件付き追加**：パフォーマンス目標達成で追加 300億ドル
- **コンピューティング提供**：Google Cloud が 5 年間で **5GW** のコンピューティングパワーを提供（さらなる追加も検討中）

## 背景と文脈

Anthropic の成長速度は驚異的で、年間収益ランレートは 2025年末の90億ドルから 2026年4月には **300億ドル超** に急伸した。

同月には Amazon も最大 **250億ドル** の追加投資を発表しており、Google・Amazon 両者からの累計投資は 1,000億ドルを超える。

### なぜ Google がここまで注力するのか

1. **Claude Code の躍進**：Anthropic の Claude Code が企業のコーディング業務で急速に普及し、GitHub Copilot や OpenAI Codex と並ぶ存在になった（[[wiki/claude-code-enterprise-adoption-2026]]）
2. **AI 競争の激化**：OpenAI と Microsoft の関係と同様に、Google も AI スタートアップと深い垂直統合を進めている
3. **Google Cloud の推進**：Anthropic のモデルは Google Cloud 上で優先的に提供されており、Cloud 事業の差別化に貢献

## 評価額の推移

Anthropic の評価額はラウンドごとに急騰しており、今回の 3,500億ドルという評価は AI 競争の過熱ぶりを象徴している。
