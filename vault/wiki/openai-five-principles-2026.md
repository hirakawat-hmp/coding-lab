# OpenAI 5つの原則（2026年4月）

2026年4月26日、Sam Altman が公開した OpenAI のガバナンス原則。「AGI が全人類に恩恵をもたらす」ための5つの指導原則を示したもの。

## 背景

OpenAI の非営利組織としての使命と営利事業の間のトレードオフに関する懸念が高まる中、CEOの Sam Altman が組織のガバナンスフレームワークを更新・公開した。外部監視の強化と民主的アクセスの促進を強調。

## 5つの原則

OpenAI が掲げるAI開発の指導原則（公式発表から）：

1. **Safety（安全性）**：AI が安全であること、人間の監視下に置かれることを優先する
2. **Beneficial（有益性）**：すべての人に利益をもたらすよう努力する
3. **Openness（開放性）**：科学的知見を広く共有し、AI の恩恵を分散させる
4. **Accountability（説明責任）**：外部監視と透明性を維持する
5. **Long-term thinking（長期的視点）**：短期利益より AGI 時代の人類の繁栄を優先する

## 意義

- Elon Musk との訴訟（「OpenAI は非営利ミッションから逸脱した」という主張）を背景に、使命への再コミットメントとして公開された
- Microsoft との独占契約改定と同日（2026年4月27日付近）という文脈での発表
