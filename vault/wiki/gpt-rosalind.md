# GPT-Rosalind — OpenAI の生命科学特化 AI モデル

![GPT-Rosalind](https://asset.watch.impress.co.jp/img/ipw/docs/2102/509/01_s.jpg)

2026年4月17日に OpenAI が発表した生命科学研究専用 AI モデル。
生物学者・創薬研究者・トランスレーショナルメディシン従事者向け。

## 利用可能チャンネル

- ChatGPT（プレビューアクセス）
- Codex
- API

## 対応ドメイン

- 分子生物学
- タンパク質工学・構造解析
- ゲノム解析・遺伝子経路解釈
- 創薬（化合物設計・バイオマーカー同定）
- 疾患生物学

## 主な機能

- 文献レビューと知識統合
- 配列解釈（DNA・RNA・タンパク質）
- 実験計画立案
- データ分析パイプラインの提案

## 性能指標

| 指標 | 結果 |
|------|------|
| GPT-5 との比較 | 生物学推論タスクで上回る |
| RNA 配列予測 | 人間専門家の 95パーセンタイルを超える |
| BixBench | 高スコア（DNA・酵素設計で顕著な改善） |

## 関連ツール

「Life Science Research Plugin」を GitHub で公開：
- Codex との連携
- 50以上の科学ツール・データソースと接続可能

## 命名の由来

Rosalind Franklin — DNA 二重らせん構造の発見に貢献した英国の化学者（1920-1958）。
