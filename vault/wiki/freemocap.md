# FreeMoCap

人間・動物・ロボットの 3D 全身運動を**完全無料・オープンソースで計測できるマーカーレスモーションキャプチャソフトウェア**。プロ向けの高価なスーツや専用機器を使わず、市販のウェブカメラ複数台で研究・アニメーション品質の動作データを取得できることを目指している。Python（PyPI パッケージ）として提供。

## マーカーレスとマーカーベースの違い

| 方式 | マーカー | コスト | セットアップ | 精度 |
|------|---------|------|-----------|-----|
| マーカーベース（従来） | 反射マーカーを体に装着 | 数百〜数千万円 | 複雑 | 非常に高い |
| **FreeMoCap（マーカーレス）** | **不要** | **無料** | **簡単** | 研究グレード |

## 技術的なしくみ

### ワークフロー

1. **カメラ較正（Calibration）**  
   ChArUco ボード（印刷した格子パターン）を全カメラの前で振ることで、カメラ間の空間関係と 3D 座標系を自動定義する。

2. **録画・インポート**  
   FreeMoCap インターフェース経由でリアルタイム録画、または既存動画ファイルを読み込む。

3. **ローカル処理**  
   CPU ベースの処理エンジンが動画を 3D 座標データに変換する。ビデオはユーザーのマシンから外部に送信されない（プライバシーファースト設計）。

### 使用技術

- **ポーズ推定**: オープンソースのコンピュータビジョン手法（MediaPipe、OpenPose 等と統合可能）
- **多視点 3D 再構成**: 複数カメラ映像を三角測量で 3D 座標に変換
- **ChArUco キャリブレーション**: OpenCV の ChArUco マーカーによるカメラ較正

## ハードウェア要件

| 構成 | 最小 | 推奨 |
|------|------|-----|
| カメラ数 | 1 台（2D トラッキング） | 3 台以上（信頼性の高い 3D） |
| カメラ接続 | USB | USB |
| 較正ツール | ChArUco ボード（印刷）| 同左 |
| GPU | 不要（CPU のみ） | 不要 |
| OS | Windows, Mac（ARM/Intel）, Linux | 同左 |

1 台のラップトップウェブカメラだけでも 2D アニメーターが使えるデータを生成できる。

## インストール

```bash
pip install freemocap
```

## ユースケース

- **バイオメカニクス研究**: 歩行・スポーツ動作の詳細分析
- **3D アニメーション**: ゲームキャラクターやアニメの動作リファレンス取得
- **理学療法・リハビリ**: 患者の動作評価と追跡
- **ロボット学習**: 人間の動作データを模倣学習に活用
- **映画・VFX**: 低予算プロダクションでのモーションキャプチャ代替
