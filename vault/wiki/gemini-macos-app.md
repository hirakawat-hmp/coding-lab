# Gemini for macOS

![Gemini for macOS](https://storage.googleapis.com/gweb-uniblog-publish-prod/images/Gemini_for_MacOS_01_Hero_16x9.width-200.format-webp.webp)

Google が 2026年4月15日にリリースした Gemini のネイティブ macOS アプリ。ウィンドウ切り替え不要で AI アシスタントを呼び出せるデスクトップ統合を実現。

## 概要

- **対応 OS:** macOS 15 以上
- **価格:** 無料（gemini.google/mac からダウンロード）
- **ショートカット:** Option + Space で即時起動
- **対象:** 13歳以上、グローバル提供

## 主要機能

### 画面共有（Screen Context）
ウィンドウを Gemini と共有し、ローカルファイルの即座の要約・洞察を取得。「ウィンドウを切り替えることなく作業を継続しながら AI の支援を受ける」設計。

### コンテンツ生成
- **画像生成:** Nano Banana モデル使用
- **動画生成:** Veo モデル使用

## 位置づけ

Google の macOS デスクトップ AI 戦略の第一歩。「これは最初のリリースに過ぎない」と将来の機能拡充を予告。

同時期、Anthropic の Claude Code Desktop も並列エージェント対応でデスクトップ機能を強化しており、デスクトップ AI アシスタント競争が激化。
