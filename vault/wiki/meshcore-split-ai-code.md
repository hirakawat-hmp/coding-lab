# MeshCore 開発チーム分裂：AI 生成コードの透明性問題（2026年4月）

メッシュネットワーク通信プロジェクト **MeshCore** の開発チームが 2026-04-23 に公式分裂を発表した事件。AI 生成コードの秘匿と商標の無断申請が原因。

## プロジェクト規模（分裂時点）

| 指標 | 数値 |
|------|------|
| 世界中のノード | **38,000+** |
| アクティブユーザー | **100,000+**（Android/iOS） |
| ファームウェアバージョン | 85+ |
| 対応ハードウェア | 75+ 変種 |
| プロジェクト開始 | 2025年1月 |

## 分裂の経緯

### トリガー 1: AI 生成コードの秘匿

チームメンバー **Andy Kirby** が **Claude Code** 等の AI ツールを使って大量のコンポーネントを開発していたが、これをチームに秘密にしていた。発覚後、Discord でのコミュニティ投票でAI生成ファームウェアへの**信頼欠如**が示された。

### トリガー 2: 商標の無断申請

2026-03-29、Andy Kirby が **MeshCore** の商標をチームに通知せず申請。内部協議が決裂し通信途絶。

## 重要な論点：AI 生成コードの透明性

この事件は「AI ツールの使用をチームに開示すべきか」という問いを提起している。

**開示を求める側の論拠**:
- セキュリティ上の懸念（AI の誤った生成や脆弱性の混入）
- ライセンス・著作権の複雑性
- チームの意思決定への影響（AI に依存した設計判断）
- 責任の所在の不明確化

**秘匿する側の論拠**:
- AI はツールに過ぎず、鍵盤配列や IDE 選択と同等
- アウトプットの品質で評価すべき

## 現在の状況

公式チームは新体制を確立:
- meshcore.io（ウェブサイト）
- docs.meshcore.io（ドキュメント）
- 新 Discord サーバー
