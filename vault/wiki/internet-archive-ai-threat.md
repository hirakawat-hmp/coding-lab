# AI の台頭がインターネットアーカイブを脅かしている

石川大樹（nomolk）が2026年4月20日に公開した考察。メディア企業が AI 学習データ対策としてクローラーをブロックした結果、インターネットアーカイブ（Wayback Machine）まで被害を受けている構造的問題を論じる。

## 何が起きているか

- NYT, The Guardian, USA Today, Reddit などの主要メディアが **Internet Archive のクローラーをブロック**
- 背景: AI 企業がアーカイブ経由で著作権コンテンツを収集することへの対抗措置
- 100人超のジャーナリストが公開書簡で抗議

## なぜアーカイブが重要か

| 用途 | 必要性 |
|------|--------|
| 政策ジャーナリズム | 政府文書の過去版参照 |
| 調査報道 | 削除・改変されたコンテンツの証拠保全 |
| 公的記録の保全 | 機関によるデータ削除への対抗 |
| 文化・歴史記録 | Web の歴史的記録 |

## 構造的な皮肉

```
AI 企業 → アーカイブ経由でコンテンツ収集
      ↓
メディア企業 → アーカイブをブロック
      ↓
インターネットの歴史的記録が失われる
```

AI から著作権を守ろうとした結果、人類の知的遺産の保存が犠牲になるという逆説。

## アーカイブが直面する他の課題

1. **動的 Web**: JavaScript 重依存サイトは完全なキャプチャが困難
2. **クローズドプラットフォーム**: Discord, Instagram, Facebook はアクセス不可
3. **有料コンテンツ拡大**: サブスクリプションモデルの普及がアーカイブを阻む

## 日本固有の問題

- 朝日・読売等の主要紙はアーカイブがほぼ存在しない
- 米国のフェアユース相当の法的保護がない → 保存活動自体が法的グレーゾーン
