# Niagara Automotive — UE5.7 自動車 VFX コース（RedefineFX）

RedefineFX が2026年4月20日にリリースした、Unreal Engine 5.7 の Niagara パーティクルシステムを使用した自動車ビジュアライゼーション向けリアルタイム VFX コース。

| 項目 | 内容 |
|------|------|
| チャプター数 | 10 |
| レッスン数 | 70+ |

| アクセス | 永続 + 将来の更新含む |

## 学習内容

- **ボリューメトリックエフェクト**: 煙、埃、砂、雪、火花
- **風・空気抵抗シミュレーション**: 外部ツール不要
- **X-Ray・爆発図エフェクト**: 商業プレゼンテーション向け
- **リアルタイム完成**: Houdini 依存なし
- 実践的な自動車モデルとシーンセットアップを含む
