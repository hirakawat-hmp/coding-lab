# Unity AI — 公式 AI 機能オープンベータ（2026年5月）

Unity が Unity 6 以降の全開発者向けに AI 機能をオープンベータとして公開。公式 MCP サーバーの提供により、Claude Code などの外部エージェントからの Unity 操作も可能になる。

## 3つのコアコンポーネント

### Unity AI Assistant
- プロジェクト統合エージェントアシスタント
- 深いプロジェクトコンテキスト認識
- 自然言語プロンプトによるプロジェクト操作

### AI Gateway
- エディタ内から好みの AI ツールへの接続
- 外部 AI エージェント（Claude Code 等）との連携
- AI Gateway 経由の利用は Unity クレジットを消費しない

### 公式 MCP サーバー
- IDE 統合向け
- Claude Code などの MCP クライアントから Unity を操作可能

## 価格構成

| プラン | AI クレジット |
|--------|--------------|
| Pro / Enterprise / Industry | 込み |
| Personal Edition | 含まれない（月$10の別サブスクが必要） |
| AI Gateway 経由（外部エージェント） | Unity クレジット消費なし |

## 意義

Unity 公式が AI 統合を正式にサポートすることで：
1. サードパーティ AI ツールとの連携が**信頼性の高いものに**
2. Claude Code などの AI コーディングエージェントから Unity プロジェクトを操作できる**新しいワークフロー**が生まれる
3. ゲーム開発・3D コンテンツ制作での AI 活用が加速
