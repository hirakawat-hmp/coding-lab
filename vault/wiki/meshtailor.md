# MeshTailor

3D メッシュに対して **UV シームを自動配置する AI ツール**。グラフベースの Transformer アーキテクチャにより、ゲーム開発などで長らく手作業が必要だったプロ品質の UV 展開を数秒で自動生成する。深圳大学・Simon Fraser 大学による研究成果（arXiv 2603.27309）。

## UV 展開（UV Unwrapping）とは

3D モデルのテクスチャは 2D 画像として管理されるため、3D サーフェスを 2D 平面に「展開」する作業が必要となる。これが UV マッピング（UV Unwrapping）。

- **U・V**: 2D テクスチア空間の軸（3D 空間の X・Y・Z と区別するため異なる文字を使用）
- **シーム（Seam）**: メッシュの切り込み線。シームに沿って 3D メッシュを「切り開いて」2D に展開する
- **UV アイランド**: シームによって分割された 2D テクスチャ上の各パーツ
- **歪み（Distortion）**: 展開時に避けられないテクスチャの引き伸ばし・縮み

シームの配置は **テクスチャの品質に直結**するため、3D アーティストはゲームキャラクターや小道具ごとに最適な位置を手動で設計してきた。

## MeshTailor のアーキテクチャ

### デュアルストリームエンコーダ

2 つの並列ストリームでメッシュを入力とする:

| ストリーム | 処理内容 |
|----------|---------|
| **トポロジストリーム**（上流） | グラフエンコーダでメッシュのエッジ接続関係を抽出 |
| **形状ストリーム**（下流） | 事前学習済みの点群エンコーダでグローバル形状を抽出 |

2 つのストリームを融合することで、**局所的な接続性**と**大域的な形状意味**の両方を捉える。

### 階層的シーム表現（ChainingSeams）

シームをコースからファインへ階層的に直列化する。

1. 全体を分割する主要シームループを予測する
2. 分割されたサブメッシュに再帰的に同じ処理を適用する
3. 意味的に一貫したパーツ分解を段階的に生成する

### ポインタ付きオートリグレッシブ Transformer

- **ポインタ層**: 局所近傍を頂点ごとにたどってシームをトレース
- **プロジェクションフリー**: 射影を介さないのでアーティファクトが発生しない
- **エッジアライメント**: 生成されたシームは必ずメッシュのエッジ上に乗る

## 従来手法との比較

| 手法 | シーム品質 | 欠点 |
|------|----------|------|
| 最適化ベース | 断片化しやすい | ギザギザの境界 |
| 外部学習ベース | 投影アーティファクト発生 | 幾何歪み |
| **MeshTailor** | プロ品質 | コードは未公開（2026-04 時点） |

## ユースケース

- **ゲームアセット制作**: 大量の 3D アセットの UV 展開を自動化しパイプラインのボトルネックを解消
- **テックアート**: アーティストの慣習に従った知的なシーム配置で品質を均一化
- **VR/メタバース**: リアルタイムレンダリング向けのテクスチャアトラス最適化
- **キャラクター・プロップ**: 有機的・ハードサーフェス両方の複雑な形状に対応
