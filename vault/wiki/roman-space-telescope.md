# ナンシー・グレース・ローマン宇宙望遠鏡（Nancy Grace Roman Space Telescope）

![ローマン宇宙望遠鏡](https://media.wired.jp/photos/69f542df02476e084f20d29c/3:2/w_2560%2Cc_limit/RST_2020_Still.jpg)

NASAが開発する次世代宇宙望遠鏡。ハッブル宇宙望遠鏡と同等の解像度を維持しながら、視野を100倍以上に拡大した「全天サーベイ型」の観測機器。

## スペック

| 項目 | 内容 |
|------|------|
| 主鏡径 | 2.4m（ハッブルと同等） |
| 視野 | ハッブルの約100倍以上 |
| 波長域 | 赤外線（0.5〜2.3μm） |
| 観測地点 | L2ラグランジュ点（地球から約150万km） |
| 打ち上げ予定 | 2026年9月初旬（当初2027年5月から前倒し） |

## 主要科学目標

### 1. ダークエネルギーの解明

数千個のIa型超新星爆発を観測し、宇宙膨張の加速度を高精度で計測する。現在の物理学が説明できない「暗黒エネルギー」の正体に迫る。

### 2. ダークマターの地図作成

弱い重力レンズ効果を統計的に解析し、宇宙全体の暗黒物質の分布マップを作成。

### 3. 太陽系外惑星の探索

コロナグラフ装置により、**既存装置比で最大1,000倍**の鮮明さで太陽系外惑星を撮影できる。約10万個の惑星発見を期待。

## 命名の背景

「ナンシー・グレース・ローマン」はNASA初の女性天文部長。ハッブル宇宙望遠鏡の「母」とも呼ばれる。
