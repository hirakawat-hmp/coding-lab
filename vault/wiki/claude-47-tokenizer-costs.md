# Claude 4.7 新トークナイザーのコスト実測

Anthropic が Claude 4.7 で導入した新トークナイザーの実コストを実測分析した報告。
公式ドキュメントが「1.0〜1.35倍」と示すのに対し、実際は最大 1.47倍のトークン増加が確認された。

## 実測値まとめ

| コンテンツ種別 | トークン増加率 |
|-------------|-------------|
| 技術文書 | 1.47× |
| 本番コードファイル | 1.45× |
| シェルスクリプト | 1.39× |
| TypeScript | 1.36× |
| 英語散文 | 1.20× |
| CJK 言語 | 1.01× |

## 性能とのトレードオフ

- IFEval ベンチマーク: 厳密な命令遵守が +5% ポイント向上
- 「より直接的な指示遵守」が可能に（サンプル数 N=20 のため限定的）

## 財務的影響

- 典型的な 80 ターンの Claude Code セッション:
  - 旧: 約 $6.65 → 新: $7.86〜$8.76（**20〜30% 増加**）
  - 基本トークン単価は変わらず

## プロンプトキャッシュへの影響

- モデル切り替え時に既存キャッシュプレフィックスが無効化
- キャッシュに保存・取得するトークン量が増加
- コールドスタートとターンごとのコスト両方に影響

## 実務的含意

Claude 4.7 に移行する際は、コスト見積もりを 1.3〜1.5 倍でバッファーを持って計算すること。
特に日本語・中国語・韓国語コンテンツはほぼ影響なし。
