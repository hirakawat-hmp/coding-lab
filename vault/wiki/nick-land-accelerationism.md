# ニック・ランドとシリコンバレー加速主義の帰還

![Nick Land](https://media.wired.jp/photos/69f603ddf98c28e0f3f410bd/master/w_2560%2Cc_limit/COPYTNY_NickLand_FINAL.jpg)

1990年代に加速主義とAI終末論を唱えた哲学者ニック・ランドが、2026年のシリコンバレーで「預言者」として再評価されている現象。

## ニック・ランドとは

- **背景:** ウォーリック大学でサイバネティクス文化研究ユニット（CCRU）を主宰
- **1990年代の主張:** 資本主義・AI・人間の時代遅れ化は不可避。技術加速により人間文明は変容する
- **衰退:** 薬物乱用・精神的危機でアカデミアから消える（1990年代後半）
- **2012年:** 「ダークエンライテンメント」発表 — 民主主義反対・AI統治による権威主義的未来を予測

## シリコンバレーでの再評価

| 人物 | 関与 |
|------|------|
| マーク・アンドリーセン | 「私のお気に入りの哲学者」と称賛 |
| サム・アルトマン | e/acc（効果的加速主義）の信奉 |
| OpenAI/Anthropic/Midjourney研究者 | SFのイベントで共演 |

## 思想の現代的解釈の変化

**元来:** 左派的加速主義（資本主義・技術の不可避な終焉への適応）
**現代シリコンバレー版:** 「AIの乗っ取りに抵抗するよりAI開発を加速させよ」 → e/acc（Effective Accelerationism）

## 中心的メッセージ（変わらず）

「認識可能な人間的なものは近い将来に何も生き残らない」

## なぜ今か

AIが急速に人間の能力を超え始める中、ランドの暗い予言が現実に近づいているように見える。若いテックワーカーが変化の意味を理解する枠組みとして使っている。

## 批判的視点

ランドの哲学はもともと政治的に危険な思想（民主主義否定・権威主義容認）を内包する。シリコンバレーでの「再解釈」は元来の思想の危険な側面を無視している可能性がある。
