# Microsoft Copilot Studio ツール設定の進化（2026年5月）

Microsoft Copilot Studioのツール機能が2026年5月時点で大幅に強化された。AIエージェントがツールを実行する際の**ユーザー確認と入力取得**の制御が細かくカスタマイズ可能になった。

## 3つの新機能

### 1. ツール実行前の確認機能

「登録操作などを実行する前にユーザーに確認を取る」設定が追加。
- 確認メッセージはカスタマイズ可能
- 変数を利用して動的にメッセージ内容を構成できる

### 2. 入力値ごとのユーザー入力要求

各入力項目について個別に確認の有無とプロンプトをカスタマイズ可能。
- 入力されない場合のエスカレーション設定もできる
- 必須/任意のきめ細かい制御

### 3. 指示文による曖昧性の対応

入力項目が形式上は満たされていても内容が曖昧な場合、AIが自動的に詳細情報を聞き返せる。
- 指示文とツール入力プロンプトの**両方**で設定することがポイント
- 「指示文のみ」では意図通りに動作しないことがある

## 設計上の意義

エージェントが単独で判断して実行するのではなく、**人間との対話を組み込んで**実行する設計が容易になった。ヒューマン・イン・ザ・ループ（HITL）の実装コストが下がる。
