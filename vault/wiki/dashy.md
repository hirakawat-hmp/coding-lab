# Dashy

![Dashy](https://i.gzn.jp/img/2026/04/25/dashy/00_m.png)

無料のオープンソース・セルフホスト型ダッシュボードツール。「ブックマークでは物足りず、監視ツールでは堅すぎる」というニーズを満たし、複数のウェブサービスをワンスクリーンで整理できる。Docker対応。

## 主要機能

- **セクション別カード整理**: サービスリンクをカード形式でカテゴリ分け
- **稼働状況確認**: サービスのリアルタイムヘルスチェック
- **ウィジェット**: RSS・気象情報・カレンダー等を追加可能
- **レスポンシブ**: モバイルにも対応
- **テーマ**: 複数テーマから選択可能
- **ユーザー管理**: Admin/User/Guestの3段階権限

## ユーザー権限

| ロール | できること |
|--------|----------|
| Admin | 全カスタマイズ権限 |
| User | 表示設定の部分カスタマイズ |
| Guest | ログインなしで公開アイテムのみ閲覧 |

## インストール

```bash
# Docker Composeで起動
docker-compose up -d
```

## 用途

- ホームラボ（Home Lab）のサービス一覧
- チームの内部ツールランチャー
- 個人の作業環境ポータル
