# OpenAI Advanced Account Security

2026年4月30日にOpenAIが発表した高度なアカウントセキュリティ機能。ChatGPT・APIアカウントの不正アクセス対策を強化する。

## 背景

2026年4月にAIプラットフォームアカウントの侵害事例が急増しており、それへの対応として機能強化。同時期にOpenAIはFedRAMP Moderate認定（2026年4月27日）も取得し、エンタープライズ向けセキュリティ強化に注力している。

## 主な機能（推定）

- アカウントの二段階認証（2FA）強化
- セッション管理の改善
- 疑わしいログインの検出・通知
- デバイス管理機能
- APIキーの不正利用モニタリング

## 位置づけ

[[wiki/claude-security]]（コードスキャン）とは異なり、アカウント自体のセキュリティ強化。AI企業が相次いでセキュリティ機能を強化している2026年4〜5月トレンドの一部。
