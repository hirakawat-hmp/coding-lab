# V-Ray for Blender Community Edition

Chaos が 2026年4月16日にリリースした V-Ray レイトレーシングレンダラーの Blender 向け無料コミュニティエディション。商用利用可能、3ヶ月ごとのライセンス更新制。

## 無料版の仕様

| 項目 | 内容 |
|---|---|
| 最大解像度 | 2,560 × 2,560 px（2K） |
| 出力形式 | 8bit のみ（EXR・32bit TIFF は不可） |
| 商用利用 | 可 |
| レンダーファーム | 不可（ローカルマシンのみ） |
| パイプライン統合 | 不可 |
| コラボ・AI ツール | 不可 |
| サポート | なし（製品直接サポートなし） |
| ライセンス更新 | 3ヶ月ごと |

## 有料版

- 月額: $33/月
- 年額: $199/年

## 意義

プロ向けレイトレーシングレンダラーとして定評ある V-Ray が Blender コミュニティに無料で開放。商用利用可であるため、フリーランサーや小規模スタジオが本格的なレイトレーシングを無料で試せるようになった。

## Blender レンダラーの競合環境

| レンダラー | 特徴 |
|---|---|
| Cycles（Blender 内蔵） | オープンソース・無料 |
| EEVEE（Blender 内蔵） | リアルタイム |
| V-Ray Community | プロ品質・無料（制限あり） |
| Arnold for Blender | Maya 系プロ向け |
| Redshift | GPU レンダリング特化 |
