# Microsoft 365 Copilot エージェント機能（2026年4月 GA）

![Microsoft 365 Copilot](https://image.itmedia.co.jp/news/articles/2604/24/yu_copilot.jpg)

Microsoft が 2026-04-22 に一般提供開始した、Microsoft 365 の Word・Excel・PowerPoint における Copilot エージェント機能。単なる「下書きアシスト」を超え、自律的な多段タスク実行が可能になった。

## 各アプリの新機能

### Word

![Word Copilot](https://image.itmedia.co.jp/news/articles/2604/24/yu_word.jpg)

- 対象読者に合わせたツール適用
- コンテンツ改善の自動実行
- ファイル内での段階的・自律的実行

### Excel

- データ分析・表作成の自動化
- グラフ生成・モデル構築
- 複数ステップのデータ操作を一括実行

### PowerPoint

![PowerPoint Copilot](https://image.itmedia.co.jp/news/articles/2604/24/yu_powerpoint.jpg)

- 既存テンプレートを保持しながら新データで更新
- プレゼンテーションの自動再構築

## 設計哲学

> 「最終的な判断はユーザーが行う」

自律実行しつつも**人間によるコントロール維持**を設計の中心に据えている。完全自律ではなく「Human-in-the-loop」アーキテクチャ。

## 利用可能プラン

| 対象 | プラン |
|------|--------|
| 法人 | Microsoft 365 Copilot、Microsoft 365 Premium |
| 個人 | Microsoft 365 Personal/Family |

## 文脈

ChatGPT（OpenAI）が Workspace Agents を発表した翌月に、Microsoft が Office 製品でエージェント機能を GA。エンタープライズ AI の「自律実行」競争が激化している。
