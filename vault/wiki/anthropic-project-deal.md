# Anthropic Project Deal

![Anthropic Project Deal](https://image.itmedia.co.jp/aiplus/articles/2604/26/yu_projectdeal.jpg)

Anthropic が2026年4月に発表したAIエージェントの市場実験プロジェクト。「エージェントが実世界の複雑な業務をどの程度処理できるか」を検証し、経済効果の測定も目的とする。

## 概要

69組の参加者が1週間にわたり Claude を使ってAIエージェントを運用する実験。Slack統合環境で、人間の監督を最小化した自律的な業務実行を検証した。

## 主要な実験結果

- **186件の課題**に対して4,000ドルの予算で完結
- Claude Opus 4.5 と Haiku 4.5 の性能差を体験的に確認
- 参加者の **46%** がサービスに「満足」と評価
- 人間の監督なしにエージェントがSlack上で自律的に業務を処理

## 意義

Anthropicは「AIエージェントが人間の要望をより正確に理解できれば、企業と個人の間の価値創造が最大化される」とコメント。企業向けエージェントサービスの経済的価値を実証しようとする取り組みで、**エージェントAIの商用化加速**を示す動きとして注目される。
