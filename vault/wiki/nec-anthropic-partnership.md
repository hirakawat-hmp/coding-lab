# NEC・Anthropic 戦略的パートナーシップ（2026年4月）

![NEC・Anthropicパートナーシップ](https://image.itmedia.co.jp/business/articles/2604/24/ea1635237_0424nec_01.jpg)

NEC が 2026年4月23日、Anthropic との「戦略的パートナーシップ」を発表した。協議開始から **3週間**（正式決定は発表の2日前）という異例のスピードで合意に至った。

## 主要合意内容

- **Claude Code 展開**：NEC グループ **3万人** に Claude Code を展開
- **業務特化ソリューション**：「Claude Cowork」を共同開発（NEC 固有業務向け）
- **投資規模**：NEC から Anthropic へ **200億円以上**
- **NEC グローバルパートナー**：Anthropic の公式グローバルパートナーとして認定

## 戦略的意義

NEC の BluStellar Scenario プラットフォームと Anthropic の Claude 技術を統合。
2030年までに約 **1兆3000億円** の収益目標を設定している。

### 「電撃的協業」の背景

ITmedia の報道によると、NEC の COO と Anthropic の COO が3週間で合意に至った。この迅速な決定は、日本の大企業としては極めて異例で、AI 競争の急速な展開が大企業の意思決定スピードを変えている象徴とも言える。

## Claude Code の企業展開

NEC グループ3万人への Claude Code 展開は、日本企業への Claude Code 大規模導入例として最大級のもの。

関連記事：「開発時間70%削減」「新機能リリース24→5日に短縮」という開発効率改善が報告されている（[[wiki/claude-code-enterprise-adoption-2026]]）。
