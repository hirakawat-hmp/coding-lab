# GPT-5.5

![GPT-5.5](https://openai.com/content/images/2026/04/gpt-5-5-hero.jpg)

OpenAI が 2026-04-23 に発表した LLM モデル。コードネームは **"Spud"**。ChatGPT および Codex にロールアウト中。

## 性能指標

| ベンチマーク | スコア |
|-------------|--------|
| Terminal-Bench 2.0 | **82.7%**（最先端） |
| SWE-Bench Pro | **58.6%** |
| レイテンシ | GPT-5.4 と同等 |

## 得意領域

1. **エージェント的コーディング**（Agentic Coding）
2. **コンピュータ操作**（Computer Use）
3. データ分析
4. ドキュメント・スプレッドシート作成
5. オンラインリサーチ
6. 初期段階の科学研究

## 特徴：エージェント的タスク実行

複雑なマルチパートタスクを渡すだけで、以下を自律的に実行:
- **計画立案**（Planning）
- **ツール使用**（Tool Use）
- **自己チェック**（Self-verification）
- **曖昧性回避**（Ambiguity handling）
- **継続実行**（Sustained execution）

従来モデルのように「各ステップを人間が慎重に管理」する必要がなくなった点が最大の進化。

OpenAI の「AI スーパーアプリ」戦略の一環。GPT-5.5 は GPT-5 系列の性能を維持しながら、エージェント機能を大幅に強化した中間バージョン。
