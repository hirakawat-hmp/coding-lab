# 1m精度標高データ + YOLO で全国の前方後円墳を自動検出

## 概要

国土地理院の DEM1A（1m精度標高データ）と YOLO11n を組み合わせて、全国の前方後円墳を機械学習で自動検出した研究（Zenn、2026年4月29日）。75件の潜在的未知古墳を発見した。

## データパイプライン

```
国土地理院 DEM1A 標高データ
    → 約146万枚の WebP タイル（合計250GB）
    → 3チャンネル画像生成（マルチヒルシェード・傾斜・曲率）
    → YOLO11n で 1,070,504 枚のタイルをスキャン
    → PAX_SAPIENTICA DB（5,245件の既知古墳）とクロスリファレンス
```

**重要技術**: 768×768px 拡張タイルを使用してタイル境界での分割問題を解決。

## 結果

| 項目 | 数値 |
|------|------|
| 潜在的未知古墳の発見 | **75件** |
| 確認済み検出 | 573件 |
| 偽陽性 | 2,344件（除外） |
| 既存DB照合 | 1,037件一致 |

## 教訓

- アルゴリズム改善より**人手レビュー**が精度向上に最も効果的
- 離散的な地形特徴を3チャンネルで表現する手法は他の地形特徴検出にも応用可能
- 公的機関の高精度オープンデータ（国土地理院）が研究の根幹

## 応用可能性

同じアプローチで:
- 全国の城址・古代遺跡の探索
- 土石流・地すべり地形の自動検出
- 農業・環境モニタリング（地形変化検知）
