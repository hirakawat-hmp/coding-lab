# Spotify Verified by Spotify（人間アーティスト認証バッジ）

Spotifyが2026年4月30日に発表した、人間アーティストとAI生成コンテンツを区別するための公式認証制度。

## 概要

| 項目 | 内容 |
|------|------|
| 名称 | Verified by Spotify |
| 発表日 | 2026年4月30日 |
| 目的 | 人間アーティストとAI生成コンテンツの識別 |
| 対象 | 人間のアーティストプロフィール |
| 除外 | AI生成コンテンツ・AIアーティスト |

## 認証要件

- Spotify のプラットフォームポリシーへの準拠
- コンサート、グッズ、SNS、カウント連携などの実績証明
- 99パーセンタイル以上の認証率基準を達成

## 背景

AI音楽生成ツールの急速な普及（Deezer 調査では日次アップロードの44%がAI生成音楽）を受け、リスナーが人間のアーティストを容易に識別できる仕組みが求められていた。Spotify による今回の措置はプラットフォームとして明確に「人間性の証明」を価値として打ち出す姿勢を示す。

テイラー・スウィフトらがディープフェイク対策として声・肖像の商標出願を行ったのと同時期の動きで、AI時代における「人間のオーセンティシティ」への需要が高まっていることを示す。
