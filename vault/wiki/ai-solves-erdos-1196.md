# AIが数学者7年の難問を80分で解く：エルデシュ問題#1196

2026年4月27日、23歳のアマチュア数学愛好家リアム・プライス氏がChatGPT GPT-5.4 Proを使い、組み合わせ論の未解決問題「エルデシュ問題#1196」を**わずか80分**で解明した。数学者ジャレド・デューカー・リクトマンが7年間追い続けた問題。

## 事件の概要

| 項目 | 内容 |
|------|------|
| 使用AI | GPT-5.4 Pro（ChatGPT） |
| 解決者 | リアム・プライス氏（23歳・アマチュア） |
| 問題 | エルデシュ問題#1196（組み合わせ論・グラフ理論） |
| 所要時間 | 80分 |
| 専門家の評価 | フィールズ賞受賞者テレンス・タオ氏が高く評価 |

## テレンス・タオ氏の評価

フィールズ賞（数学のノーベル賞）を受賞したテレンス・タオ氏が：

> 「ほぼ自律的にAIがエルデシュ問題を解いた**最初の事例**」

と高く評価。数学研究の転換点として記録される可能性がある。

## 「Book Proof」とは

数学者リクトマン自身がXで：

> 「これはまさに『Book Proof（神の書物の証明）』だと思った」

と投稿。「Book Proof」とはエルデシュが提唱した概念で、「神の書物に記されているような、最も美しく簡潔な証明」を指す。

## 示唆：AIと数学の新しい関係

この事例が象徴するもの：

1. **学術的障壁の民主化**: アマチュアがAIを使い専門家7年分の研究を追い越せる
2. **直感と形式の融合**: 「美しい証明」の感覚をAIが体現できるか
3. **数学研究の変革**: 未解決問題へのAIファーストなアプローチが加速
