# Claude Opus 4.6 → 4.7 システムプロンプト変更点

Simon Willison による Claude Opus 4.6 と 4.7 のシステムプロンプト比較分析（2026年4月18日）。

## 安全性・行動の変更

### 強化された制限
- 子どもの安全に関する指示が `<critical>` タグで囲まれ、より厳格なガイドラインに
- 摂食障害関連の新ガイドライン：特定の栄養・運動数値を提供しないよう制約
- 物議を醸すトピックへの「はい/いいえ攻撃」（単純二択を迫る誘導）への防御

## ツール・機能の追加

### 新しいエージェントスキル
- **PowerPoint エージェント**が追加（既存の Chrome、Excel に加え）
- `tool_search` メカニズム：「できない」と言う前に利用可能なツールを確認
- ツールを使って曖昧さを解消する新しいメカニズム（ユーザーに確認を求める前に）

## ユーザー体験の改善

| 変更点 | 内容 |
|--------|------|
| 押しつけがましさの軽減 | 会話終了の意思を尊重するようになった |
| 簡潔さの強調 | 「ユーザーを圧倒しないよう集中した応答を保つ」 |
| 先行確認の削減 | 最初に確認を求めるより、合理的な仮定で行動することを優先 |

## 削除された要素

- 絵文字・アスタリスクを避ける旧指示（モデルが自然に改善済みのため不要に）
- トランプ政権2025年に関する特定の言語（知識カットオフが2026年1月に更新）

## 解釈

このプロンプト変化が示すもの：

1. **透明性の文化**：Anthropic がシステムプロンプトを公開・比較されることを前提に設計している
2. **イテレーティブな安全性改善**：問題が発見されるたびにプロンプトレベルで対処
3. **自律性と安全性のトレードオフ調整**：より自律的（ツール活用・確認削減）にしつつ、リスク領域は厳格化
