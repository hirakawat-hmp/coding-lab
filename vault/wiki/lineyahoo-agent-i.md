# LINEヤフー Agent i

![Agent i](https://image.itmedia.co.jp/business/articles/2604/21/ea1635237_260421lyai_01.jpg)

LINEヤフー株式会社が2026年4月20日に発表した AI エージェントサービス。LINE と Yahoo! JAPAN の両プラットフォームに統合され、ChatGPT・Gemini との差別化として「実際のユーザーサービスとデータベース」を強みとして打ち出す。

## 主要数値

| 指標 | 数値 |
|------|------|
| LINE アクティブユーザー（2025年12月時点） | 100万人超 |
| Yahoo! JAPAN ユーザー | 150万人超 |
| 2026年中に提供予定のエージェント数 | 20以上 |

## 機能

- カスタマーサポート自動化
- 業務レポート分析
- ファイナンシャルプランニング支援
- LINE Official Account AI Mode
- LINE AI Support
- Agent i Biz（マーケティング向け）

## 競合優位性

CPO のコメント: 「ChatGPT や Gemini などの汎用 AI エンジンと異なり、LINEヤフーには実際のユーザーサービスとデータベースがある」

100 以上のサービスエコシステム（LINE・Yahoo! ショッピング・Yahoo! ファイナンス等）との深い統合が差別化ポイント。

## 位置付け

日本のメガプラットフォームがグローバル AI 競争に対して、ローカルデータアドバンテージで対抗する戦略の典型例。
