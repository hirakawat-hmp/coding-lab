# ArcBrush v1.0 — ノードベース非破壊画像編集ツール

Albert Omoss作のノードベース画像編集ツール。75種のノードによるグラフ型パイプラインで非破壊ワークフローを実現。基本機能は無料・アカウント不要。オプションのAI機能はクレジット制で提供。Windows/macOS対応のネイティブC++デスクトップアプリ。

## 主要機能

| 機能 | 詳細 |
|------|------|
| ノード数 | 75種（カスタマイズ可能） |
| 非破壊編集 | 全操作が後から編集可能 |
| GPU加速 | 高速処理 |
| カラーパレット | 無限バリエーション生成 |
| バッチエクスポート | 6フォーマット + スプライトシート |
| グラフ共有 | .arcbファイル形式 |
| AIオプション | クレジット制 |

## 位置づけ

[[wiki/davinci-resolve-photo]]（写真向けカラーグレーディング）や[[wiki/hitem3d-2]]（3Dアセット生成）とは異なり、デザイナー向けの汎用ノードベース画像処理ツール。After EffectsのExpressionsやBlenderのGeometry Nodesに近い操作概念を持つ。
