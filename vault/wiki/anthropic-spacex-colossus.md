# Anthropic × SpaceX Colossus 1 コンピュート契約

![Higher usage limits for Claude and a compute deal with SpaceX](https://www.anthropic.com/images/news/higher-limits-spacex.jpg)

Anthropic が SpaceX の Colossus 1 データセンターへのアクセス権を取得したと2026年5月6日に発表。22万台以上の NVIDIA GPU と300メガワット超のコンピューティング能力を活用し、Claude のレート制限を大幅に引き上げた。

## 主な変更点（即時適用）

| 変更内容 | 対象 |
|---------|------|
| Claude Code レート制限を2倍に | Pro / Max / Team / Enterprise |
| ピーク時間制限を撤廃 | Pro / Max（Claude Code） |
| API レート制限を大幅増 | Claude Opus 系モデル |

## インフラ拡大戦略

Anthropic は SpaceX 以外にも複数のコンピュート調達を進めている：

- **Amazon**: 最大5ギガワット（2026年末までに約1GW）
- **Google + Broadcom**: 5GW（2027年開始）
- **Microsoft + NVIDIA**: $300億 Azure 容量
- **Fluidstack**: $500億インフラ投資

## xAI との関係

同日、イーロン・マスクが xAI を SpaceX の AI 部門「SpaceXAI」に統合すると発表。Colossus 1 の運営主体が変わる可能性があるが、Anthropic との既存契約への影響は不明。→ [[wiki/xai-spacex-ai-division]]

## 国際展開

規制産業（金融・医療・政府）向けに地域データ所在地準拠を必要とする国際展開を強化。米国データセンターの電気料金上昇分のカバーも約束。
